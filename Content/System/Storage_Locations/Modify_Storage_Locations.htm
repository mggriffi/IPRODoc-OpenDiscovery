Modify Storage Locations

You have the ability to rename a storage location after it has been created. To edit the name of a storage location:

  1. Click the Settings icon in the top-right corner of the screen. The Settings icon is a global button that displays in every module of the platform.

  2. The System Manager opens. In the left pane of the System Manager, click Storage Locations.

  1. Click on the appropriate station location in the list. The Location Details display on the screen.
  2. Click the Edit icon in the top-right corner.
  3. Make required changes and click Save.
  4. Repeat these steps to edit other storage location names.
  5. Notify Media Manager users of pertinent changes.
