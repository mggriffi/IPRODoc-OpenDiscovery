Delete a Security Template

Complete the following steps to delete a security template:

  1. Click the Settings icon in the top-right corner of the screen. The Settings icon is a global button that displays in every module of the platform.

  2. The System Manager opens. In the left pane of the System Manager, click Security Templates.

  1. Click on the name of the security template you would like to delete. The template profile opens on the right side of the screen.

  2. Click the Delete icon .
  3. A confirmation message appears. To continue with the delete, select Delete.
