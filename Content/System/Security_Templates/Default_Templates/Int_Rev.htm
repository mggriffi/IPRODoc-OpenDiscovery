Internal Reviewer Template

The permissions outlined in the following table are the default permissions assigned in the Internal Reviewer Permissions template. The table lists the following:

  • Permission Name
  • Definition
  • Linked Permission - Some permissions are dependent on other permissions. When permissions are linked, you must have both the permission and it's linked permission to get access to the functionality.

Click on the following items to expand a table describing the permissions associated with that section:

Note: This template has no permissions assigned for Media & Data or Reports.
