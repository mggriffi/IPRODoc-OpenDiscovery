Modify a Security Template

Complete the following steps to modify the permissions of a security template:

  1. Click the Settings icon in the top-right corner of the screen. The Settings icon is a global button that displays in every module of the platform.

  2. The System Manager opens. In the left pane of the System Manager, click Security Templates.

  1. Click on the name of the security template you would like to edit. The security template profile opens on the right side of the screen.

  2. Click the Edit icon . You are now able to modify the permissions for the selected template.

  3. Select each needed section (Case Management, Media and Data, Search and Review, Reports, or System), then click the check box beside each permission you would like to add to the template. You can also remove a permission from the template by clicking on a check box that has already been selected. For more information on the permissions available for selection, as well as how they link together, see Permissions List.

  4. Click Save.

  5. Repeat these steps for all security templates needing to be edited.
