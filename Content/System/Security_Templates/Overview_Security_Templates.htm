Overview: Security Templates

Security templates allow you to apply a pre-configured set of permissions to a group, or multiple groups, which you can then tweak as needed. The templates allow easy assignment of specific permissions. Permission changes made in are reflected in .

Security templates can only be applied when creating a new group. For more information on how to accomplish this, seeCreate New GroupsCreate New Groups. For existing groups, permissions must be assigned manually.

Security template management is located in the System Manager, accessed from the gear icon at the top-right corner of the screen.

From here, you have the ability to create, modify, or delete security templates. System Manager allows you to manage these templates as described in the following procedures as described in the following lessons.
