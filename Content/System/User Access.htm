Manage User Access

User access is determined by permissions assigned to the group to which the user belongs. Users who do not have permission to access a specific component of , such as Media Manager, will see the Media Manager icon grayed out on the homepage.

To successfully log in to the , a user must be at least one of the following:

  • Super Admin
  • In a group that has any of the following permissions:
    • Media Manager
    • Self-Service
    • Reporting
    • Review

Provide needed permissions for the user by setting those permissions at the group level and assigning the user to that group. To learn more about user and group management, see Manage Users and Groups. For more information about the permissions available in , and how they're linked together, see Permissions List.
