Modify Copy Stations

To edit the details for a copy station:

  1. Click the Settings icon in the top-right corner of the screen. The Settings icon is a global button that displays in every module of the platform.

  2. The System Manager opens. In the left pane of the System Manager, click Copy Stations.

  1.  From the list, click the appropriate Copy Station that needs to be edited. A page opens showing the station details.
  2. Click the Edit icon in the top-right corner of the screen. The fields displayed on the Station Details page become modifiable.
  3. Make needed changes and click Save.
  4. Repeat these steps to edit details for other copy stations.
  5. Notify Media Manager users of pertinent changes.
