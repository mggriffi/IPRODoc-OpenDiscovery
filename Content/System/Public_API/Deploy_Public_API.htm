Deploy Public API

What's in the package?

The tar.gz file contains docker images for both Public API service and the Public API gateway (Kong). All of the files needed to configure and deploy are in this package.

Where to deploy?

In theory, anywhere a docker container can run could work, as long as the containers will have access to the IPRO upstream services via HTTP/HTTPS (IIS and Processing box).
