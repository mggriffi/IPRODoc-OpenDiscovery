Delete Users

To delete a user from the user database (from all products):

  1. Click the Settings icon in the top-right corner of the screen. The Settings icon is a global button that displays in every module of the platform.

  2. The System Manager opens. In the left pane of the System Manager, the Users tab opens by default.

  1. Click on the name of the user you would like to delete. The user profile opens on the right side of the screen.

    Note: You can search for all or part of the user’s first or last name.

  2. Click the Delete icon .
  3. A confirmation message appears. To continue with the delete, select Delete.

Deleted users who are logged in to any product will remain active until they log out (or the product times out). After that point, they will not be recognized by any product.
