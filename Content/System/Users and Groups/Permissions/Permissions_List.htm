Permissions List

This topic covers the full list of permissions that can be assigned in . Permissions are assigned at the group level and filter down to the users within the group. The permissions outlined in the following tables are all of the permissions available to be configured. The table lists the following:

  • Permission Name
  • Definition
  • Linked Permission - Some permissions are dependent on other permissions. When permissions are linked, you must have both the permission and it's linked permission to get access to the functionality. For example, if you are given permission to Production Shield, but you don’t have the Case Administration expander permission, you will not see Production Shield in your application.
