Manage OPEN DISCOVERY Environment Settings

Environment Settings enable communication between Media Manager, copy stations, and applications. Additional settings include those for Self-Service and bar codes.

Use the following procedures to configure settings:

  • Change Environment/Share Settings
  • Define Media Attachment File Path
  • Define Elasticsearch Endpoint
  • Define Barcodes
  • Sign In With Multi-Factor Authentication (MFA)
  • Configure Single Sign-on (SSO)
  • Configure LIVE EDA
