Define Elasticsearch Endpoint

It is necessary to define your existing Elasticsearch endpoint in order to use the optional System Monitoring function. If you do not have Elasticsearch, please contact IPRO Support for information about downloading, installing, and configuring this open source application. Additional information is available at https://www.elastic.co/.

To define the Elasticsearch endpoint, do the following:

  1. Click the Settings icon in the top-right corner of the screen. The Settings icon is a global button that displays in every module of the platform.

  2. The System Manager opens. In the left pane of the System Manager, click Environment Settings.

  1. Click the edit icon in the top-right corner of the screen to modify the fields on this page.
  2. In the Elastic Search Endpoint field (under Configurations), enter the URL configuration address. For example, http://es-endpoint001.es-sam.local:9200 .Only http or https are allowed.
  3. Click the corresponding Save button.
