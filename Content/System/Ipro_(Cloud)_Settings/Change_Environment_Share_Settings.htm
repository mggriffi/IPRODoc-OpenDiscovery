Change Environment/Share Settings

The Application Environment for Integration Service is defined during installation. It enables communication between Media Manager, copy stations, and applications. It is also used for Self-Service Uploads to be processed into Review.

The Upload Network Share location must be defined for groups who are assigned the System permission Upload Self-Service and who will be uploading files for later processing (process into review is not available unless the Case permission “Self-Service Upload & Process” is set). This is the network location to which the files will be copied.

To make changes to either or both of these settings:

  1. Click the Settings icon in the top-right corner of the screen. The Settings icon is a global button that displays in every module of the platform.

  2. The System Manager opens. In the left pane of the System Manager, click Environment Settings.

  1. Click the edit icon in the top-right corner of the screen to modify the fields on this page.

  2. To change the Application Environment for Integration Service:

    1. Click and select the needed Processing environment. Check with your administrator if you are not sure of the environment to select.
    2. Click the corresponding Save button.
  3. To define or change the Upload Network Share location:  
    1. Enter a new location or correct an existing entry. Use UNC format (for example, \\server001\SelfService\FileUploads). Full access rights to this directory are required.
    2. Click the corresponding Save button.
