System Administration FAQs

This topic provides answers to frequently asked questions regarding the Administration component.

Do I need to create contacts ahead of time?

No, contacts can be created on the fly in Self-Service or Media Manager.

Do I need to create a copy station if my data is already on the network?

No, your data is already on the network so a copy station is not needed.

Do my paths need to use a UNC format or can I use a drive letter?

Your paths need to use the UNC format.
