Export Numbering Options

Bates Numbering

This area of the form allows you to define the way in which you want exported/produced documents numbered and defined in your review case. You can create a new Bates number or continue previous numbering for a rolling production.

  • Create new numbers: Re-number pages/documents using the remaining options in this section of the form.

  • Use image priority: Select one or more document sets from which documents will be selected.

Production Numbering Fields

This area of the form allows you to define production fields in your review case. These fields must be completed if you want to see the Production numbering within the Results Grids fields in your review case. Production Numbering options are visible to users only if they have the Production permission.
