Create a Pick List

A pick list is a set of items (terms) from which a reviewer can choose when editing a field in Record View.

To create a pick list:

  1. In the Pick Lists panel on the left side, click . The fields appear on the right.

  2. In the Name field, enter a name for the list.

  3. In the Terms field, enter items to comprise the list, one item per line. Maximum size for each term is 1,000 characters.

  4. When finished, click Save.
