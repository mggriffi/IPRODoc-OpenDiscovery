Create Keyword Lists

Keyword searching can assist you with document management, review, and other tasks. You create and schedule a keyword list for processing.

  1. In the Set Keyword List Details window, enter the below:

    1. List Name:(up to 85 characters)

    2. Notes:(up to 500 characters). Alphanumeric, spaces, and/or underscore characters can be used.

    3. Search Scope: Keywords can be scoped to an entire case or a specific search.

      Note: The Keyword List Name and the Search Scope cannot be changed once created. If you select Saved Search, the Keyword List will need reprocessing if the saved search is modified.

  2. Click Next.
