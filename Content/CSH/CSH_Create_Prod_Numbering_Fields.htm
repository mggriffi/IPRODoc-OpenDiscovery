Create Production Numbering Fields

To select numbering options:

In the export set form, click to open the Production Numbering Fields if this area of the form is not visible. Complete each section as needed.

    Option

    Description

    Begin Bates

    Requires fields for storing beginning- and/or ending-production numbers. Select these fields: PRODBEGDOC; PRODENDDOC

    End Bates

    Begin Attachment Bates

    Requires fields for storing beginning- and/or ending-production numbers for attachments. Select these fields: PRODBEGATT; PRODENDATT

    End Attachment Bates

    Custom Text Field

    If your case includes a field for storing production-related notes, select the field in which the notes should be stored.

    Custom Text

    If you select a custom text field, enter notes for the production in this field.
