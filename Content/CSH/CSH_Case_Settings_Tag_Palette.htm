Create a Tag Group and Tags

The Tag Palette includes groups of tags that are to be applied to each document in your case. This is where you define and manage tag groups and tags for each case.

  1. The left panel displays the list of tag groups already created. To create a new tag group, select the plus icon at the top of the list, beside the search bar.

    1. Enter the tag group name (up to 100 characters), and then click Save.

    2. Select required tag group rules.

  2. Create the group's tags as follows:

    1. Click Create Tags.

    2. In the Tag Group area, enter the group’s tags (up to 250 characters for each name).

    3. When all tags have been entered, click Add.
