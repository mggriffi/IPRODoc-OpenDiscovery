Export Output Location Options

This area of the form allows you to define the types of files that will be produced and how they will be organized in your export media. This is where you will set up your native files, images and text. If you are producing a load file, the sub-section "Output Locations" is where you will set up the load file options and fields.

In the export set form, click to open the Output Locations if this area of the form is not visible.
