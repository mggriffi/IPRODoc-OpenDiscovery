Set Keyword Scheduling

  1. In the Set Keyword Scheduling Settings enter the following:

    1. Timezone

    2. Select Frequency

    3. Auto save results after processing.

    Option

    Description

    Occurs immediately

    The list will be processed one time, immediately after the list is completed.

    Occurs once at (time)

    Run the list automatically (if case changes have occurred) once a day at the specified hour.

    Occurs every... Starting/Ending at (times)

    Run the list automatically.

    Enter or use the up/down arrows to select the interval, and enter the starting and ending hours or click to select them.

    Auto save results after processing

    Select this option if you want to create the keyword tags and field after the keyword list is processed. You can also create them separately, at a later time.

  2. Click Save.The Keyword List will be created. The progress can be monitored in the Job Manager.
