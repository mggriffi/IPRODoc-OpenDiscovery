Review Reports

Administrators can generate case-specific Review reports. This page lists the review-based reports available in the Reports module.

If an activity on which a report is based has not occurred in Review, the report will have no data. For example, if no Technology Assisted Review (TAR) projects have been created in Review , then TAR reports will have no data.

For details on the functionality upon which processing reports are based (such as data extraction), see the appropriate topic in Review.

 

The following Review reports are available:

Analytics

REPORT

Description

Case Data Summary Report

Summarizes case size details such as number and total size (in GB) of files (image, native, and text files).

TAR Project Certification Details Report

Presents detailed results for the certification round, including precision data for the review in terms of how accurately non-responsive documents have been identified by the TAR process.

TAR Project Discrepancies Report

Identifies documents for which there is a disagreement between the reviewer and the TAR analysis regarding document responsiveness.

TAR Project Score Distribution Report

Presents distributed category scores (per validation round) for all rounds in the TAR project.

TAR Project Summary Report

Provides an overview of the TAR project, with data ranging from the total number of documents in the TAR project (based on the category index selected for the project), to projected Responsive and Non Responsive documents for the project.

TAR Validation Details Report

Presents detailed results for each validation round, including precision data for the review in terms of how accurately Responsive documents have been identified by the TAR process.

Production

Report

Description

Production Deliverable Report

Presents details on production and imported production history jobs.

Production Summary Report

Summarizes production and imported production history jobs completed in Eclipse Administration.

Redacted Documents Report

Summarizes redaction activity for a selected case.

Redaction Detail Report

Includes detailed information on redacted documents as defined by your administrator.

Quality Control

Report

Description

Gap Report

Identifies missing pages, that is, breaks in the numbering sequence of image keys (case, production, or other document numbers) for a particular case.

Review Status

Report

Description

Privileged Report

Presents documents based on tags selected for the report. Included are document totals, the applied tag(s), and field data and other applied tags as defined by your administrator.

Review Status by Custodian Report

Presents review status of documents in a particular review pass, sorted by a field selected by your administrator that contains custodian details. Tag details may also be included if configured by your administrator.

Summary by Custodian Report

Summarizes review activities (number of documents reviewed, not reviewed, etc.) sorted by a field selected by your administrator that contains custodian details.

Reviewer - Metric

Report

Description

Review Status by Reviewer Report

Includes review status of documents in a particular batch, sorted by assigned reviewers. Tag details may also be included if configured by your administrator.

Reviewer Edits Per Day Report

Lists all reviewers by user name and the number of distinct actions (field editing and/or tagging) each reviewer has made on each day of the date range, for a specific date range. Can be useful in evaluating review efficiency.

Reviewer History Report

Summarizes reviewer activities (such as field editing and tagging) for a selected period of time, including total hours spent by each reviewer, number of edits, and edits per hour and day.

Reviewer History Details Report

Provides detailed information on reviewer activities (such as field editing and tagging) for a selected period of time, including day-by-day summary of time spent and editing details.
