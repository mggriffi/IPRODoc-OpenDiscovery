Processing Reports

Administrators can generate case-specific Processing reports. This page lists the processing-based reports available in the Reports module.

If an activity on which a report is based has not occurred in the Processing module, the report will have no data. For example, if data has not yet been extracted for a case, then the Data Extract Item Summary report will have no data.

For details on the functionality upon which processing reports are based (such as data extraction), see the appropriate topic in eCapture by IPRO.

The following Processing reports are available:

Ingestion

Report

Description

Data Extract Deduplication Summary Report

Provides a summary of items not processed due to de-duplication settings.

Data Extract Detailed Error Report

Provides a summary of errors encountered during processing.

Data Extract Filter Manager Summary Report

Provides results of rules applied and detail on how they affected the final total of promoted documents.

Data Extract Item Report

Provides a summary of errors encountered during processing.

Discovery Summary Report

Provides a summary of items discovered by item type and category.

Source Container Report

Provides a list of the top level containers for selected case.

Streaming Deduplication Detailed Report

This report contains detailed de-duplication and indicates the original file a document was a duplicate of.

Streaming Deduplication Overlay Report

The report queries all documents, including duplicates, in Process and Data Extract Jobs that are part of the Exports; not just processed documents in the Export Jobs. The report returns results for all original documents and duplicates for the selected Export Job(s)

Export

Report

Description

Export Series Summary Report

Provides results of all exports within a series based on Export Jobs and their associated Volumes. Additional information is provided that breaks down the exported documents by sizes and type. Files not exported are also broken down by type along with any error information associated with exports within the series.

Media

Report

Description

Language Summary Report

Provides a breakdown of identified languages grouped by file type and further by language.
