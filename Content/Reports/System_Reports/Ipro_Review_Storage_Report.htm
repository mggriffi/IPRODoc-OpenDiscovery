IPRO Review Storage Report

Reports the total size of all records in the selected Managing Client's case database(s) that were loaded within the selected date range. End Date will represent the most recent date that terminates the desired date range. Default aggregation is by Case, Client, and Managing Client. All parameters are optional. For details about the various components displayed in the report, see Report Details below.

Note: To access this report, select System Reports from the Select Case drop-down menu:

 

Complete the following selections:

Report Title: A report title is selected by default; change the title if needed.

Managing Client: Select this option to view processing data for a specified Managing Client.

Start/End Dates: Select the starting and ending dates for the time period of interest by clicking in each field and selecting dates from the resulting calendars.

Code Name Level Granularity: Select this option to view processing data organized at the code name level.

Import Date Level Granularity: Select this option to view processing data organized by import date.

When ready, click Run Report.

Report Example

Report Details

Component

Description

Current Database Size

The current size of the case SQL database. This value is not date range dependent.

Current Search Index Size

The current size of the case dtsearch index. This value is not date range dependent.

Date Range

If no date range is specified, the default date range becomes the dates of the first and last case imports.

Docs With Images

The count of documents that have associated images.

Image File Size

The sum of file sizes of images associated with case documents.

Image Page Count

The count of images associated with case documents.

Native Size

The sum of all of the native sizes reported in the case database.

Natives

The count of documents that have an associated native file.

Prod Image Size

The sum of file sizes of produced images.

Prod Page Count

The count of pages produced.

System Size

The sum of Native Size, Image File Size, Prod Image Size, Current Search Index Size, and Current Database Size

Text Count

The count of documents whose text field is not null

Text Size

Sum of text size stored in the case database for all documents.
