Reports FAQs

 

Can I create custom reports?

Administrators have the ability to create custom reports that are implemented into Reporting. Please contact for additional assistance in report creation.

I’m trying to run a report, but it’s not opening.

Ensure that your pop-up blocker is disabled.

Can I change the report preview image from the default image?

Yes, you can replace the current preview image. See Customize the Report Preview Image .
