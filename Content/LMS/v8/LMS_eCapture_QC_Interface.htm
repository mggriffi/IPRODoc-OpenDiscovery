Understand the eCapture QC Interface

The following sections outline each of the various areas of the QC user interface and provide links to topics related to the specific tasks performed in the application.

QC Layout

The QC window and layout of tabs and work areas is flexible and can be changed to meet your requirements. You can:

  • Split tabbed windows into multiple windows
  • Auto-hide windows
  • Move windows around the desktop or to a secondary monitor.

After you create a desired layout that you would like to use repeatedly or share with other QC users, you can save the layout to an .INI file and reload it later or share it with the other users. You can also restore the default layout without losing your saved layout(s).

For more information about working with the QC window and layouts, see Save the QC Interface Layout.

Image View Area

The Image View area has three tabs: Image, Thumbnails, and View.

Image Tab

The Image tab shows the output from the file processing. Some of the tasks you can complete on the Image tab include:

  • Fit the image in the window
  • Rotate the image
  • Zoom in/out on the image
  • OCR the current page
  • Decrease the image quality

To view a table that describes each of the buttons available on the Image tab, see Overview eCapture QC.

Thumbnails Tab

The Thumbnails tab window allows you to view thumbnail-sized images of documents shown in the Documents pane. This is particularly useful for multi-page documents.

For more information, see Thumbnails Tab Window.

View Tab

The View tab displays the document image. There are no additional options on the View tab. For a Data Extract Job, only the View tab appears.

For more information, see View Tab Window.

Documents Window

The Documents window allows for multiple Processing Job and/or Data Extract Job sessions. Each session is defined by selecting categories and flag conditions (Passed QC, Exceptions, and so on). The right-context menu options in the Documents window allow you to:

  • Create an Export Set. For more information, see Create an Export Set.

  • Search through the current session, Job, Custodian or Case (Project). For more information, see Conduct a Session Search.

  • Refine an existing search. See Conduct a Session Search.

  • Sort two or more column fields. For more information, see Modify Columns and Column Fields in the Documents List Window.

  • Run a query on the selected item’s column field (e.g., author) and select from: Equals to, Does Not Equal, Contains, or Does Not Contain. The system then returns all the documents that match the selected criteria. In addition, an Advanced Search can be conducted where you can further narrow a search. For more information, see Conduct a Session Search.

  • Set or clear a flag for all selected items. For more information, see Set or Clear a QC Flag for Selected Documents.

  • Sort the selected column field in ascending or descending order and modify existing fields. For more information, see Modify Columns and Column Fields in the Documents List Window.

  • Open the Output directory for a document. For more information, see Open the Output Directory for the Selected Document.

  • Close a session. For more information, see Close a Session or Search Tab.

Session Tabs

When you initially create a session for a Processing Job or Data Extract Job, you have the option to create more than one. For each session you create, a corresponding tab exists with the conditions you selected. For example, you may have selected the Category Microsoft Word, and then chosen the Passed QC flag. When you mouse over the session tab, it will show this information. When you right-click a session tab, a context menu appears.

On the right-click context menu, you can :

  • Search through a session's, Job, Custodian, or Case (Project) and refine your search. For more information, see Conduct a Session Search.
  • Sort the column fields. For more information, see Modify Columns and Column Fields in the Documents List Window.
  • Close the session. For more information, see Close a Session or Search Tab.
  • Create Export Sets. For more information, see Create an Export Set.

QC Flags

The QC Flags are located in the Flags pane on the right side of the interface. The Flags pane contains a list of the existing QC flags and makes it easy to select a QC flag for an extremely large set of documents at once without impeding performance.

For more information, see Work with the QC Flags Window.

Metadata and Extracted Text/OCR Pane

In the Metadata and Extracted Text/OCR pane, you can view metadata for a selected document as well as the OCR text extracted for the document.

Metadata Tab

The Metadata tab displays the Process Status field, which indicates the status of the QCed item.

Extracted Text Tab

The Extracted Text/OCR tab displays the OCR text for the selected document.

Note: The Extracted Text/OCR tab is not available for Cloud Imaging Jobs.

QC Options

The QC Options dialog box allows you to save the QC layout to an INI file or load a previously-saved layout. Saving settings and loading settings allows for consistency when QCing is performed by several QCers.

For more information about saving QC Options, see Save QC Options to an INI File.
