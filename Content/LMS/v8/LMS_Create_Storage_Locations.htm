Create Storage Locations

To define storage locations, proceed through the following steps. For best results, plan/use a consistent naming convention for presenting all aspects of a storage location (building, room, shelf, etc.).

  1. Click the Settings icon in the top-right corner of the screen. The Settings icon is a global button that displays in every module of the platform.

  2. The System Manager opens. In the left pane of the System Manager, click Storage Locations.

  1. Click . The Create New Storage Location page opens.  
  2. Enter the storage location name and click Save:

  3. Repeat these steps to create other storage locations.
