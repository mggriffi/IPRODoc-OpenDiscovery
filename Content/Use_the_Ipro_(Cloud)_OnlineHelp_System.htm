Common Browser Options

For best results, maximize the size of your browser window. If this is not possible, scroll bars are available that allow you to scroll and view the full topic content.

Pages can be refreshed using your browser’s Refresh command. This may reset the page to display its default settings. You can also press ALT + Left Arrow to navigate to the previous page displayed.

Topic Navigation

When reviewing a topic, there are some topic elements that can make it easier to navigate the Help system.

Button

DESCRIPTION

Return to Module Home Page

When viewing a topic, you can return to the home page for the module by clicking the module icon in the top-right corner of the topic window. For example, clicking this icon in the top-right corner of a Self-Service topic, returns you to the Self-Service home page.

 

Find Topic in Table of Contents

At the top of each topic is a breadcrumb that indicates where in the table of contents a topic is located. You can click the blue link to navigate back to the home page for the module area the topic belongs to.

View Glossary Term Definition

In topics, glossary terms are displayed in a bold and italicized font. When you hover over a glossary term, a pop up window displays with the definition for the glossary term.

View Full-Size Images
