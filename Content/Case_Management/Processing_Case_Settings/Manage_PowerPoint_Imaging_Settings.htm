Manage PowerPoint Imaging Settings

You can set and modify PowerPoint imaging settings for a Processing case in . These settings, which are also found in , offer you the flexibility of customizing how PowerPoint files are imaged when ingested into your case. Settings modified in are updated in , and vice-versa.

Follow the instructions below to learn how to modify PowerPoint imaging settings in , as well as to review detailed definitions for each setting.

Note: There are additional imaging settings available in . The settings found in represent the most commonly modified settings. For further customization options, use the app. See Create a Streaming Discovery Job for more information.

View and Modify PowerPoint Imaging Settings

To view and/or modify PowerPoint imaging settings:

  1. Open Case Management and locate the Processing case whose settings you would like to view or modify.

  2. Click the hamburger icon corresponding to the needed case.

  3. From the menu that appears, select Case Settings.

  4. Click the Imaging tab at the top of the Case Settings work area.

  5. On the Imaging page, you can view a selection of General, Word, PowerPoint, and Excel imaging settings. To view and/or modify the full set of PowerPoint imaging settings available in , click the Manage PowerPoint button.

    Note: You can access this button by scrolling down to the bottom of the PowerPoint card, or by selecting the PowerPoint link on the right side of the Imaging page.

  6. On the PowerPoint page, update settings as needed. See the table below for more information about these settings.

  7. When finished, click the Save button in the top-right corner of the page. If you exit without saving, any changes you made will be lost.

    Note: You can discard any changes you have made by clicking the Cancel button in the top-right corner of the page, or by exiting the page.

Understand PowerPoint Imaging Settings

Review the table below to learn more about the various settings you can update for PowerPoint imaging in .

Setting

Definition

Convert using Outside In (Stellant)

 

When selected, Outside In (Stellent) is used to process images for PowerPoint files. Select this option to:

  • Allow for faster and more consistent generation of images on the first pass

  • Reduce the amount of time spent manually QCing these document types

Full metadata is extracted and time zone imaged output reflects the time-zone handling options configured for the Processing Job. All files processed by Outside In (Stellent) receive the Stellent Processed flag in QC.

Original settings (As Is)

Select this option to use the existing formatting settings saved in the native PowerPoint file.

Note: Additional formatting settings on this page become disabled when this option is selected.

Slide Size

Choose a slide size or As Is from the drop-down menu.

Paper Size

Select an output paper size or As Is from the drop-down menu.

Presentation View

Select the view to use during image conversion. The options are:

  • Slides

  • Outline

  • Notes Pages (notes and slide on one page)

  • Notes Pages Split (notes and slide on separate page)

  • Handouts

Page layout orientation

Select the appropriate orientation option. The option you select determines how the system orients pages for viewing:

  • As is

  • Portrait

  • Landscape

Slide orientation

Select the appropriate orientation option. The option you select determines how the system orients slides on the page for viewing:

  • As is

  • Portrait

  • Landscape

Print hidden slides

Select this option to print slides that are hidden from the slide show.

Print comments

Select this options to print comments for your slides.

Frame slides

Selecting this option prints a border around each slide.

Fit to page preference

Select the Scale to Page option to ensure all available text appears on the slide.

Include Linked Content Summary

Select this option to ensure that the data collected includes hyperlinks and OLE linked files. If any linked content exists in a document, a QC flag is added.

Header and Footer (As displayed on slides)

Determine how to display content in the header and footer of slides. The options include:

  • Date and Time

    Select Date and Time if you want the page header to list the date and time at the top of the image.

    • Update Automatically - Select this option to have the page header display either the Date Last Saved or the Date Created at the top of the image. Then select a format option for the date and time.

    • Fixed - select this option if you want to manually enter a fixed date and time in the image header.

  • Footer

    Select Footer if you want a footer at the bottom of the image.

    If Footer is selected, do one of the following:

    • Check As Is to maintain the existing footer for the slide, or

    • Enter static text that you want printed at the bottom of the image.

  • Show title slide

    If Footer is selected, choose a Show Title Slide option to define whether to show the footer on the title slide image. The options are: As is, Show, Do not show.

  • Slide Number

    If Footer is selected, choose a Slide Number option to define whether a slide number should show on the image. The options are: As is, Show, Do not show.

Header and Footer (As displayed on Notes and Handouts)

Determine how to display content in the header and footer of the page. These options are disabled when using the "Slides" Presentation View. The options include:

  • Date and Time

    Select Date and Time if you want the notes/handouts to list the date and time.

    • Update Automatically - Select this option to have the page header display either the Date Last Saved or the Date Created at the top of the image. Then select a format option for the date and time.

    • Fixed - select this option if you want to manually enter a fixed date and time in the image header.

  • Header

    Select Header if you want a header at the top of the image.

    If Header is selected, do one of the following:

    • Check As Is to maintain the existing headers, or

    • Enter static text that you want printed at the top of the image.

  • Footer

    Select Footer if you want a footer at the bottom of the image.

    If Footer is selected, do one of the following:

    • Check As Is to maintain the existing footer for the slide, or

    • Enter static text that you want printed at the bottom of the image.

  • Page Number

    If Footer is selected, select the Page Number option to define whether or not a page number should show on the image. The options are : As is, Show, Do not show.

Handout Display

Determine how to display content when utilizing the "Handouts" Presentation view:

  • Slides per Page

  • Order (if generating 4 or more slides per page)
