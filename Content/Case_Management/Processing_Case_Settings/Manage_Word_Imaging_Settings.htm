Manage Word Imaging Settings

You can set and modify Microsoft Word imaging settings for a Processing case in . These settings, which are also found in , offer you the flexibility of customizing how files are imaged when ingested into your case. Settings modified in are updated in , and vice-versa.

Follow the instructions below to learn how to modify Word imaging settings in , as well as to review detailed definitions for each setting.

Note: There are additional imaging settings available in . The settings found in represent the most commonly modified settings. For further customization options, use the app. See Create a Streaming Discovery Job for more information.

View and Modify Word Imaging Settings

To view and/or modify Microsoft Word imaging settings:

  1. Open Case Management and locate the Processing case whose settings you would like to view or modify.

  2. Click the hamburger icon corresponding to the needed case.

  3. From the menu that appears, select Case Settings.

  4. Click the Imaging tab at the top of the Case Settings work area.

  5. On the Imaging page, you can view a selection of General, Word, PowerPoint, and Excel imaging settings. To view and/or modify the full set of Word imaging settings available in , click the Manage Word button.

    Note: You can access this button by scrolling down to the bottom of the Word card, or by selecting the Word link on the right side of the Imaging page.

  6. Update settings as needed. See the table below for more information about these settings.

  7. When finished, click the Save button in the top-right corner of the page. If you exit without saving, any changes you made will be lost.

    Note: You can discard any changes you have made by clicking the Cancel button in the top-right corner of the page, or by exiting the page.

Understand Word Imaging Settings

Review the table below to learn more about the various settings you can update for Microsoft Word imaging in .

Setting

Definition

Convert using Outside In (Stellant)

 

When selected, Outside In (Stellent) is used to process images for Word files. Select this option to:

  • Allow for faster and more consistent generation of images on the first pass

  • Reduce the amount of time spent manually QCing these document types

Full metadata is extracted and time zone imaged output reflects the time-zone handling options configured for the Processing Job. All files processed by Outside In (Stellent) receive the Stellent Processed flag in QC.

Show Hidden Text

Select this option to see hidden text, if any, contained in Word documents.

Paper Size

Use the drop-down menu to select the output paper size for Word documents during image conversion.

Track Changes

Select the appropriate revision option. The option you select determines how the system handles revisions within Word documents.

  • As is - Print the document as it is according to the Office Settings on the computer.

  • All Markup - Print the document with revisions shown.

  • No Markup - Print the document with no revisions shown.

  • Both Copies - Documents are printed. If a document has revisions, it's printed again with the revisions shown. Documents with revisions will then have two sets of images, one right after the other.

Page layout orientation

Select the appropriate orientation option to determine how pages should be displayed. Select the appropriate orientation option. The option you select determines how the system orients images of Word documents.

  • As is

  • Portrait

  • Landscape

Fit to page preference

Select the Scale to Page option to scale the contents of the page to fit in the printable area. This sets the PrintZoomPageWidth and PrintZoomPageHeight to the paper size of the printer when printing Word documents.

Date field handling

Determine how to present date fields (aka auto date) when encountered:

  • Replace with Date Created - will replace with creation date.

  • Replace with Date Last Saved - will replace current date with last saved dated.

  • Replace with Comments - displays the Date Field Comments field where you can enter the text that should replace the contents of the date field.

  • Replace with Field Code

  • Do Not Replace - will not replace the date (e.g. Macros)

  • Remove - removes the codes entirely.

Filename field handling

Determine how to present filename fields (aka auto filename) when encountered:

  • Replace with Filename (no path) - inserts the unqualified filename

  • Replace with File Path - inserts the fully-qualified path of the original file

  • Replace with Comments - displays the Filename Comments field where you can enter the text that should replace the filename

  • Replace with Field Code

  • Do Not Replace
