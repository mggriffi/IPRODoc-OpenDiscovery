Manage Placeholder Settings

You can configure custom placeholders for documents that are being produced natively.This customization allows you to manage how placeholders are displayed, as well as what file types are automatically placeholdered during imaging of documents. The sections in this article document the process of creating, editing, and removing custom placeholders.

Create a Placeholder

To create a custom placeholder, follow the steps below:

  1. Open Case Management and locate the Processing case whose settings you would like to view or modify.

  2. Click the hamburger icon corresponding to the needed case.

  3. From the menu that appears, select Case Settings.

  4. Click the Imaging tab at the top of the Case Settings work area.

  5. On the Imaging page, scroll down to the Placeholder card. Click the Manage Placeholders button.

    Note: You can access this button by scrolling down to the bottom of the Placeholder card, or by selecting the Placeholder link on the right side of the Imaging page.

  6. The Placeholder work area opens. Click Create New Placeholder.

  7. On the first step of the Create New Placeholder wizard, define placeholder details.

    Option

    Description

    Placeholder Name

    Provide a name for the placeholder. The placeholder name is the text that displays on the image indicating that the document has been placeholdered.

    Font Family

    Select the desired font from the options available in the Font Family field. When you select the field, the full list of fonts appears in a dropdown. Scroll through the list of fonts, or search for the font in question by typing the name in the field.

    Enable Max Page Threshold

    Select the Enable Max Page Threshold slider and indicate a threshold value (1 to 10000) to limit the number of pages produced by larger files. By default, this option is disabled. When set, all pages imaged up until the threshold is reached are included in the document.

    Predefined Placeholder Image

    If you already have a placeholder image created, you can upload it on this step of the wizard. To use a predefined placeholder:

    • Click Upload Image to display the Open dialog box.

    • Select a placeholder image. Images must be in .png, .jpg, or .jpeg format with a size of 8.5 x 11 inches or 2500 x 3300 pixels.

    • Click Open. The selected image displays in the preview box.

    Placement Options

    In the Placement Options box, select the placement settings for the placeholder.

    1. Set the Alignment positioning for the placeholder.

      • Vertical Alignment: Determines placement along the vertical axis. Options include Top, Center, or Bottom. Center is the default.

      • Horizontal Alignment: Determines placement along the horizontal axis. Options include Left, Center, or Right. Center is the default.

    2. Set the left and right Indentation for the placeholder. This setting determines the horizontal spacing to the left or right of the page margins.

    3. Set the Truncation for the placeholder. Truncation determines how many characters display in the placeholder name. For example, inputting a value of 10 displays only the first 10 characters of the placeholder name and omits the rest. The default value is 128 characters.

  8. When you have finished configuring options on the first step of the wizard, click Next.

  9. On the second step of the wizard, select file types you would like the placeholder applied to when encountered during imaging. When the job runs, it creates placeholders for only those file types that are selected. By default, all file types are unselected.

    To select needed file types:

    • Click Clear all to clear the selections and individually select the desired file types.

    • Expand the various categories to view the associated file types. To expand categories, select the icon beside the category name. You can also use the search bar to filter the results. By selecting the checkbox beside the category name, every file type within that category is also selected.

    • Click Select all to select all file types.

      Note: If all items have been selected, a Deselect All button becomes available. Click this option to clear all selections.

    • Click Load Defaults to revert the selection back to the default file types.

  10. To add specific file extensions to the placeholder definition, click the Enter new file extension field and type in the extension. Click the plus icon to add it to the list. The file extension you submitted appears in the File extensions box.

    Note: You can type in the extension either with a period (".") or without one. recognizes both as valid.

  11. To import a list of file extensions from a CSV file, click Load from file. Select the CSV file and click Open. The specified files appear in the File extensions box. The CSV file may contain extensions with or without a "." (period).

    Note: If you want to remove a specific extension from the list, hover your mouse over the extension in the File extensions box and select the X button. To remove all file extensions, select the Clear All button.

  12. When all file types and extensions have been set, click Save to create the new placeholder.

Edit a Placeholder

  1. Open Case Management and locate the Processing case whose settings you would like to view or modify.

  2. Click the hamburger icon corresponding to the needed case.

  3. From the menu that appears, select Case Settings.

  4. Click the Imaging tab at the top of the Case Settings work area.

  5. On the Imaging page, scroll down to the Placeholder card. A list of configured placeholders appears in this section Click the Manage Placeholders button.

  6. The Placeholder work area opens. Locate the placeholder you would like to modify from the list. Select the Edit icon to the right of the placeholder name to open the Edit Placeholder wizard.

  7. On the first step of the Edit Placeholder wizard, update placeholder details as needed.

Delete a Placeholder

To delete a placeholder, follow the steps below:

  1. Open Case Management and locate the Processing case whose settings you would like to view or modify.

  2. Click the hamburger icon corresponding to the needed case.

  3. From the menu that appears, select Case Settings.

  4. Click the Imaging tab at the top of the Case Settings work area.

  5. On the Imaging page, scroll down to the Placeholder card. A list of configured placeholders appears in this section Click the Manage Placeholders button.

  6. The Placeholder work area opens. Locate the placeholder you would like to delete from the list. Select the Delete icon to the right of the placeholder name.

  7. A message displays asking if you would like to proceed with the removal. Click Delete Placeholder. The placeholder is permanently deleted.
