Manage Case Detail Settings

You have the ability to view and modify case details for a Processing case in . Follow the instructions below to learn how to manage these settings.

To view and/or modify case details:

  1. Open Case Management and locate the Processing case whose settings you would like to view or modify.

  2. Click the hamburger icon corresponding to the needed case.

  3. From the menu that appears, select Case Settings. The work area opens with the Case Details tab already selected.

  4. On this page you can view your case configuration. To modify case details, click the Manage Case Details button.

  5. Update settings as needed. You have the ability to update any of the following options:

    Note: Modifying and saving case details overwrites the existing settings and may affect existing processing jobs.

    Option

    Description

    Case Name

    Click into the Case Name field and type in the new name for the case.

    Code Name

    The code name allows for integration with other Processing, Review, or cases that use the same code name. Click the Code Name field and select an existing code name from the dropdown menu. Alternatively, select the plus icon beside the field to add a new code name. The field clears for you to type in the new code name. When you save your changes, the new code name is added to the Case Management start page.

    Matter Number

    Click into the Matter Number field and update the matter number as needed.

  6. When finished, click the Save button in the top-right corner of the page. If you exit without saving, any changes you made will be lost.

    Note: You can discard any changes you have made by clicking the Cancel button in the top-right corner of the page or by exiting the page.
