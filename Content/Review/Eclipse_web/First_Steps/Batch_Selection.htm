Open a Batch

To open a batch:

  1. Open the needed case as explained in "Case Selection."

  2. Click Batches in the Case View pane; the batches assigned to you will be listed. (System administrators will see all batches in the case.) Continue with step 3 or step 4.

  3. To select the next available batch assigned to you:

    1. Click Get Next Available Batch.

    2. If multiple review passes exist, select the needed review pass and click OK.

    3. Click Documents in the Case View pane. The next available batch is available in the Batch tab.

    4. Skip to step 5.

  4. To open a specific batch:

    1. If many batches are listed in the Batching tab, you may want to sort or filter the list to find the needed batch. See "Filter or Sort the Batch List" for details.

    2. In the Batching tab, click the Active option corresponding to the needed batch as shown in the following figure. Once the option is clicked, the entry changes to

      .

    3. Click Documents in the left pane. The records comprising the active batch will be available in the Batch tab.

  5. All documents initially display in the Batch tab. Select batch-specific documents by choosing one of the view options in the DOCUMENTS toolbar, as shown in this figure. If needed, click to refresh the list of documents.

  6. For best performance, maximize the browser window. For even more working space, use the browser’s full screen mode by pressing the F11 key.

  7. If this is your first experience with Eclipse, take a tour of the product as explained in "Interface Overview."

  8. Work with your administrator to learn how you will be using Eclipse. Your administrators will provide case-specific instructions.

  9. Before working in a batch, read case alerts, instructions, and any other documents provided by your administrator. See "Administrative Alerts" for details.

Filter or Sort the Batch List

It may be useful to sort or filter the list of batches so that you can evaluate and/or select from a specific set of batches.

Filter the List

  1. Log in to Eclipse and open the needed case.

  2. In the Case View pane, click Batches. The Batching tab opens.

  3. Locate the column by which you want to filter the list and click the funnel button as shown in this figure, then complete step 4 or step 5 as needed.

  4. To select from existing column entries, click one or more options in the top part of the Filter dialog box.

  5. To define more complex criteria, select filtering criteria in the bottom of the Filter dialog box:

    1. Select a needed criterion from the first list. Several options exist, such as Is Equal To, Starts With, or Does not contain.

    2. In the field under the first criterion list, enter the details by which you want to filter.

    3. If the entry in step b includes text for which capitalization is important, enter the details using the needed case and click the button.

    4. If you want to include a second criterion, select the needed connecting operator, And or Or, and repeat steps a – c in the bottom part of the dialog box. This figure shows filtering by a particular range of batch size.

    5. When the filtering criteria is defined, click Filter.

    6. To filter by an additional column(s), repeat this procedure for the additional column(s).

  6. To clear filtering:

    1. Locate the filtered column and click .

    2. In the Filter dialog box, click Clear Filter.

    3. Repeat to clear other columns.

Sort the List

  1. Log in to Eclipse and open the needed case.

  2. In the Case View pane, click Batches. The Batching tab opens.

  3. Locate and click the column heading by which you want to sort. The column will first sort in ascending order. Sorting is indicated by a small triangle above the column name.

  4. To change to descending order, click the heading again.

  5. To clear sorting, click a third time.
