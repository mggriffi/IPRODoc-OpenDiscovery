Navigating Through Annotations

To navigate through a transcript’s quick marks, highlights, and annotations:

  1. Log in to Eclipse and open the needed case or batch.

  2. Access the Document Details pane and click the Transcripts tab.

  3. Select the Auto Annotate option on the Transcript toolbar.

  4. Open the needed transcript and click Annotations.

  5. Use the navigation bar in the Annotations pane, shown in this figure.

  6. Review highlighted transcript content as well as details in the Annotations pane.

Modify an Annotation

After adding quick marks, highlights, and/or annotations, make changes as follows:

  1. Log in to Eclipse and open the needed case or batch.

  2. Access the Document Details pane and click the Transcripts tab.

  3. Open the needed transcript and take the following actions as needed.

  4. Quick marks: to remove a quick mark, simply click the highlighted line number. The quick mark will be deleted.

  5. Highlights: to remove a highlight (and associated issue tags, notes, and links), right-click the highlight and select Delete.

  6. Issue tags, notes, links: to change issue tags, notes, or links:

    1. Click Annotations in the left pane of the Transcripts tab.

    2. Click the needed highlight and take any of the following actions.

    3. Select or clear issues.

    4. Edit or delete note content. Use common keyboard shortcuts to cut, copy, or paste note text.

    5. Click Remove Link to remove a document link.

Add a Quick Mark

A “quick mark” is a simple place marker – a highlight of line numbers in the transcript’s margin, as shown in the following figure. In addition to providing visual cues, you can generate a report that summarizes quick marks.

To add a quick mark, click line numbers of the transcript content you want to mark.

Highlight Transcript Content

To highlight transcript content as shown in the previous figure (without any annotations):

  1. Log in to Eclipse and open the needed case or batch.

  2. Access the Document Details pane and click the Transcripts tab.

  3. Open the needed transcript and go to the needed page and line(s).

  4. Drag across the text to be highlighted.

Add Annotation Details

Annotation details include issue tags, document links, and/or notes. To add annotation details to highlighted transcript content:

  1. Log in to Eclipse and open the needed case or batch.

  2. Access the Document Details pane and click the Transcripts tab.

  3. If desired, select the Auto Annotate option on the Transcript toolbar to make the Annotations pane (shown in this figure) active automatically as you work.

  4. Highlight content as described in the previous procedure or click an existing highlight to select it.

  5. If you did not select Auto Annotation in step 3, right-click the highlight and select Annotate. Complete the following steps as needed.

  6. Issues: To add an issue tag, click relevant issue(s). Scroll in the list to view all issues.

  7. Document Links: Complete step 8 or step 9 to link a case document or an external file to highlighted text.

    Note: One case document or one external file can be associated with each transcript highlight.

  8. To link a case document:

    1. Make sure the needed highlight is selected in the Transcript tab.

    2. Locate and click the needed document in the Documents tab (or Search Results, etc. tab) as shown in the following figure.

    3. In the transcript Annotations pane, click Link Selected Document. The selected document appears in the Annotations pane as shown here.

    4. When reviewing highlighted text, select a highlight and click  to view the linked document in a Search Results tab.

  9. To link a file:

    1. Make sure the needed highlight is selected in the Transcript tab.

    2. In the transcript Annotations pane, click Link External Document.

    3. In the Open dialog box, navigate to and select the needed document. This document will be copied from its source location to an Eclipse location defined by your administrator.

    4. When reviewing highlighted text, select a highlight and click the External Document button, , to open and view the linked document.

  10. Notes: In the Notes area of the tab, enter needed information by typing or pasting (press Ctrl+V to paste).

  11. Repeat steps as needed for other transcript content.
