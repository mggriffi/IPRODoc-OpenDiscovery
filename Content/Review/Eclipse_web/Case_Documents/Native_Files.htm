Working with Native Files

About Native Files

If your case includes native files such as mails, spreadsheets, and/or word processing documents, you can view them in one of two ways—with the native application (if you have it) or in the Eclipse Native Viewer tab.

The Native Viewer tab in Eclipse utilizes Oracle Outside-In Technology to present native files. Over 500 file types are supported, ranging from common Microsoft applications to legacy files. See Supported Native File Types.

Native Viewer Activities

The following document activities are possible in the Native Viewer tab:

  • Viewing, including:

    • Viewing the document text in various formats (normal, draft, and preview)

    • Viewing search results and persistent highlights (if they have been configured by your administrator)

    • Finding a term(s) within the document

    • For spreadsheets: showing/hiding hidden cells

  • Creating a TIFF image from the native file (if no image file exists).

  • Copying text

  • Opening the file in its native application (if you have the native application)

  • Printing the native file

  • Adding Eclipse Analytics category examples from native file content
