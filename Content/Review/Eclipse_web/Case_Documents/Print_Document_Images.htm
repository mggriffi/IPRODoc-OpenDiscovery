Print Document Images

To print document images:

  1. For a selected record, open the Image tab as explained in Accessing Case Documents.

  2. On the Image tab toolbar, click .

  3. In the Print dialog box Options tab, select the needed Printer and Paper Source. As needed, click Properties to select printer-specific options.

  4. To evaluate the size of the print job, click Calculate at the bottom of the Options tab. The total number of pages displays to the right of the button.

  5. In the Page Range section, select the pages to be printed, all, current, or a unique set of pages. In the latter case, separate individual entries (individual pages and/or page ranges with commas). Indicate a page range with a dash. For example: 1, 3, 5-10, 15-20.

  6. To include a watermark (text only) in the print job, enter the needed text in the Watermark field. This text will appear “behind” existing document content, similar to a stamp.

  7. To add image keys to the bottom right-hand corner of each page, select the Image Key option in the Quick Endorsements area.

  8. Reviewers’ annotations (embedded text, highlighting, and/or mark-ups) will be printed by default if they exist. To omit annotations in the print job:

    1. To remove all annotations, clear the Print Annotations option.

    2. To remove some annotations, click each one in the Annotations area. These annotations will be removed from the list in the Print dialog box.

    3. To add annotation(s), click in the Annotations area and select those annotations you want to print.

  9. Redactions that have been applied in images will be printed in solid format. If you have sufficient permissions, you can select which redactions will be included in the print job and how they will appear:

    1. To remove all redactions, clear the Print Redactions option.

    2. To remove some redactions, select the Print Redactions option, then click (clear) the unwanted redaction(s). These redactions will be removed from the list in the Print dialog box. The following figure shows an example of print selections for redactions.

    3. To add redaction(s), click in the Redactions area and select those redactions you want to print.

    4. To change redaction format, select the needed format from the list below the redactions list, In the previous figure, the Crosshatch redaction format is being selected.

  10. A slip sheet can serve as a cover sheet for your print job. To define a slip sheet, click the Slip Sheets tab and select the content to be included. See "Slip Sheets tab" for details.

  11. To include document content (field data, tag information, or custom messages) in the header and/or footer of the document being printed, click the Endorsements tab and select the content to be included. See "Endorsements tab" for details.

  12. When all aspects of the print job are defined, click Print. If a color printer is being used, all colors will be maintained.
