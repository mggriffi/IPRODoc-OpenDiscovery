Printing Extracted Text

To print from the Extracted Text tab:

  1. For a selected record, open the Extracted Text tab as explained in "Accessing Case Documents."

  2. On the tab’s toolbar, click .

  3. In the Print dialog box:

    • Click Setup to select a printer and printer-specific options.

    • Click Options to define page options such as font type/size and header text. Note: the header text includes the job name (which can be customized) and page number.

    • Complete additional details in the Print dialog box and click OK. Extracted text and highlighting (if it exists) will print.
