Working with Extracted Text

About Extracted Text

As its name implies, extracted text is the text from a particular document—in plain text format. Extracted text appears on the Extracted Text tab in Eclipse. If your Eclipse database does not include a field defined as EXTRACTEDTEXT, or that field does not contain data, then the Extracted Text tab will be blank.

Extracted Text Activities

The following document activities are possible in the Extracted Text tab:

  • Viewing, including:

    • Viewing the document text in various formats (normal, draft, and preview)

    • Viewing search results and persistent highlights (if they have been configured by your administrator)

    • Finding a term(s) within the document

  • Copying text

  • Printing extracted text

  • Adding Eclipse Analytics category examples from extracted text

Adding Example Content for Categories

If Eclipse Analytics have been configured for your site, content or document examples can be added for Analytics Categories from the Extracted Text tab. See "Adding Example Content for Categories" for details.
