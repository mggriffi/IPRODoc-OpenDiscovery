Export a Report

To export a report to a .CSV or .XLS (Microsoft Excel) file:

  1. Run the report as explained in the previous procedure.

  2. At the bottom of the Reports tab, select Export to CSV or Export to Excel in the actions list and click Run.

  3. In the Save As dialog box, navigate an appropriate location for the file, enter a file name, and click Save. It is not necessary to include the file extension (.CSV or .XLS).

  4. Open either type of file in Excel or another spreadsheet program.

Note: When opening the .XLS file, you may receive a message regarding the file format. If so, respond to the message by clicking Yes and following any additional Excel prompts.
