Search History

Working with Search History

When a history search is run, results from the initial search are presented in the Search Results tab (a new search is not performed). Links for each search are also included in the Documents tab, as explained in "Returning to the Documents tab."

To run a history search:

  1. In the Documents tab, click the search of interest. Alternatively, go to step 2.

  2. To use the Search History list:

    1. Click Advanced Search in the Case View pane.

    2. Click Search History in the Advanced Search area. If the Case View pane is narrow, expand its width and/or use the arrow buttons to scroll until the Search History tab is visible, as shown in this figure.

  3. Click a search of interest. Quick searches are listed simply by the number of results, such as (719). Advanced searches that have names (whether saved or not) are listed by name.

    Search results display in place of the Documents tab.

Combine Historical Searches

You can combine two or more historical searches using common Boolean operators. To combine historical searches:

  1. Open the case being searched.

  2. Click Advanced Search in the Case View pane.

  3. Click the Search History tab.

  4. In the list of searches, note the numbers of the searches to be combined.

  5. In the Query field, enter a search expression in the following format:

    Search# AND|OR|NOT Search# AND|OR|NOT Search#...
