Run and/or Save a Search

To run and/or save an advanced search (defined as previously described):

  1. After reviewing your search definition in the Advanced Search tab, complete step 2 or step 3.

  2. To run the search without saving it, click Search Only.

  3. To save the search:

    1. Enter a unique name in the Name field.

    2. Select the folder in which the search should be stored. If the folder does not exist, create it as explained in "Search Folders."

    3. Click Save and Search to save and run your search, or click Save to save it for use at another time.

  4. If you run the search, evaluate the outcome in the Search Results, Image, Native Viewer, and/or Extracted Text tabs. Note: If no results exist, the Search Results tab will be blank.
