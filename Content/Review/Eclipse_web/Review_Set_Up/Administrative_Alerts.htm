Administrative Alerts

Your administrator will create and manage messages and documents to assist you in reviewing the documents in your case. Documents are maintained in the Eclipse system directory and available in Eclipse in the Case Details Dashboard. In addition, case alerts can be configured to notify you of special circumstances, such as scheduled system downtime.

The following table lists the types of information and/or files your administrator may include. If needed, ask your administrator about the information available in your Eclipse cases.

Files

typical uses

Case alerts

May be either short notices - typically used to notify users of unscheduled or unusual situations, or to separate documents.

Case instructions

Intended for the following types of documents:

  • Procedures and information needed by users reviewing a case.

  • Details specific to the batches in a particular review pass.

Case documents

Any documents pertinent to a case that are not discovery documents may be provided here. May include detailed case alert information.

Read Administrative Alerts

To read short administrative notices in Eclipse:

  1. Log in to Eclipse and open the needed case or batch.

  2. If the Case Details Dashboard is not visible, click Case Details Dashboard in the Case View pane.

  3. In the Case Details Dashboard menu, click Case Alerts. Or, click on the Case Alerts title bar. In either case, the alerts pane will become the primary pane in the Case Details Dashboard.

  4. As needed:

    • Enlarge the Eclipse workspace area to better view alert information (for example, maximize the browser window or unpin slideout panes).

    • Drag the right edge of the Alert column heading to make the column wider.

    • Click a column heading to sort alerts by that column.

    • Click Refresh to ensure the latest alerts appear.

  5. Read all new alerts.

Opening Administrative Documents

Note: Administrative documents may be any type of file. To view a file, the corresponding program must reside on your computer. If you cannot open a particular file, contact your administrator.

To open an administrative document in Eclipse:

  1. Log in to Eclipse and open the needed case or batch.

  2. If the Case Details Dashboard is not visible, click Case Details Dashboard in the Case View pane.

  3. In the Case Details Dashboard menu, click the file type of interest—Case Documents or Case Instructions. Or, click of the corresponding pane's title bar. In either case, the selected pane will become the primary pane in the case Details Dashboard.

  4. As needed:

    • Enlarge the Eclipse workspace area to better read document titles (for example, maximize the browser window or unpin slideout panes).

    • Drag the right edge of the File Name column heading to make the column wider.

    • Click a column heading to sort file details by that column.

    • Click Refresh to ensure the latest files appear.

  5. In the selected pane, locate the needed file (scroll if needed) and click . The file opens in its native program. If your system cannot find the program, you may be asked to find it. Follow the system instructions to locate the needed program.

  6. Read and/or print the file as needed. When finished, close it. If you are asked to save changes, click No.
