Edit Text or Sticky Note Annotations

If you have sufficient permissions, edit embedded text or sticky note contents on an image in the Image tab as follows:

  1. On the Image tab toolbar, click .

  2. In the image area, locate and double-click the embedded text or sticky note to be edited.

  3. In the Embedded Text or Sticky Note dialog box, modify the text as needed. For embedded text, change font characteristics if needed (see "Add Embedded Text").

  4. When finished, click the Save and Close button.

Change Mark-Up Characteristics

If you have sufficient permissions, change the characteristics of a mark-up object (line or shape) in the Image tab as follows:

  1. In the image area, locate the mark-up to be changed.

    Tip: If a line is too thin to be effectively clicked, increase the size of the image as explained in "Resize an image."

  2. On the Image tab toolbar, click .

  3. Left-, then right-click the mark-up and select Edit.

  4. In the Markup dialog box, change the color, width, and/or opacity level. Evaluate and adjust your changes in the dialog box as needed.

  5. To revert to the original settings, click the Undo button.

  6. When finished, click the Save and Close button.

Resize Highlights

Highlights can be resized. If you have sufficient permissions, you can change the size of highlights in the Document Detail’s Image tab as follows:

  1. In the image area, locate the highlighting to be resized.

  2. On the Image tab toolbar, click .

  3. Click the highlight to select it, then drag an edge or a corner until the highlight is the needed size.

  4. If needed, re-adjust the location of the highlight by placing the mouse pointer, , in the center of the redaction and dragging it.

  5. When finished, click or another button to stop editing highlights.

Move Annotations

In the Document Detail’s Image tab, highlights, embedded text, and sticky notes can be moved. If you have sufficient permissions, move any of these items as follows:

  1. In the image area, locate the item to be moved.

  2. On the Image tab toolbar, click and click the item. The mouse pointer changes to .

  3. Place the pointer over the item that needs to be moved; drag it to the needed location, then release the mouse button.

  4. When finished, click or another button to stop the move activity.

Delete Annotations

Important! No warning message or Undo function exists when deleting annotations. Ensure that the intended annotation is selected before you delete it.

All annotations in the Document Detail’s Image tab can be deleted. If you have sufficient permissions, delete annotations as follows:

  1. On the Image tab toolbar, click .

  2. Left-, then right-click the annotation and select Delete.
