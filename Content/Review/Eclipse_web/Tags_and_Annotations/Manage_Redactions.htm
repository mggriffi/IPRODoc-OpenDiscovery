Change Redaction Style

If you have sufficient permissions, you can change redaction style (for example, to cross hatch or outline so that you can view the redacted content without removing the redactions).

A selected style applies to all redactions for the current user during the current session (or until you change the style).

To change redaction style:

  1. For a selected record, on the Document Details pane’s Image tab toolbar, click next to the Redaction button.

  2. Select the desired redaction style from the second menu—Solid, Outline, or Cross Hatch.

Change Redaction Category

If you have sufficient permissions, you can change the redaction category (color/label) as follows:

  1. For a selected record, in the Document Details pane’s Image tab, locate a redaction to be to be changed.

  2. On the Image tab toolbar, click .

  3. Left-, then right-click the redaction and select the needed redaction category from the list.

  4. When finished, click or another button to stop the redaction activity.

Resize a Redaction

If you have sufficient permissions, change the size of a redaction as follows:

  1. For a selected record, in the Document Details pane’s Image tab, locate a redaction to be resized.

  2. On the Image tab toolbar, click .

  3. Click the redaction to select it, then drag an edge or a corner to resize it.

  4. If needed, adjust the location of the redaction by placing the mouse pointer, , in the center of the redaction and dragging it.

  5. When finished, click or another button to stop the redaction activity.

Move a Redaction

If you have sufficient permissions, move a redaction as follows:

  1. For a selected record, in the Document Details pane’s the Image tab, locate a redaction to be moved.

  2. On the Image tab toolbar, click .

  3. Place the mouse pointer in the center of the redaction and drag it to the needed location, then release the mouse button.

  4. When finished, click or another button to stop the redaction activity.

Delete a Redaction

Note: No warning message or Undo function exists when deleting redactions. Make sure of the selection before deleting.

If you have sufficient permissions, delete redactions as follows:

  1. For a selected record, in the Document Details pane’s Image tab, locate a redaction to be to be deleted.

  2. On the Image tab toolbar, click .

  3. Left-, then right-click the redaction and select Delete.

  4. When finished, click or another button to stop the redaction activity.
