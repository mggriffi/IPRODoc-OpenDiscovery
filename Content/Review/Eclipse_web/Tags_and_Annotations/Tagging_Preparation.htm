Tagging Preparation

Before you tag documents and/or document pages:

  1. Review this overview to ensure an understanding of how tagging works in Eclipse. Also make sure that you know:

    • case guidelines, including the coding form to use and associated rules (if they exist). For information, check the Case Alerts and Instructions folders. See "Administrative Alerts" and/or contact your administrator.

    • how to work in the case table. See "The Case Table" for details.

  2. Log in to Eclipse and open the needed case.

  3. In the Case View pane, click Documents.

  4. Optional: Display a specific subset of documents by performing a search, opening a Smart Folder, filtering on one or more columns, or selecting an Analytics cluster or category.

  5. Batches only: If you are working in a batch, select the scope of listed documents in the Documents from list.

    Tip:  A document subset can be very useful for applying a tag(s) to multiple documents.

  6. Ensure that all Eclipse workspace components are in a convenient location. See "Basic Layout Changes" for details on customizing the workspace.

  7. In the Record View pane, select the needed coding form to access the needed tag display.

  8. If no tags appear in the Document Tags tab:

    1. Right-click the Private Tags heading and select Tag Display > Modify Tag Display.

    2. Select tags you need for review and click Apply.

  9. Document tagging only:

    1. Also in the Record View pane, enable tagging by clicking edit or pressing Ctrl+F2.

    2. If auto-saving is configured for the form and you want to save changes automatically, click actions and select the Auto Save option. (Auto-saving applies to both field edits and tagging.)

      Tip: Auto Save is not available for coding forms that include rules.

  10. In the Document Tags (or Page Tags) tab, review the tags and the rules defined for each tag group. Rules are summarized at the top of each tag group, as shown in this example.

  11. Based on your permissions and the case/batch needs, add public tags as explained in "Working with Public Tags."

  12. If private tags (visible and used by only you) will help you organize your work, see "Working with Private Tags."

  13. If desired, simplify the Document and/or Page Tags tabs:

    1. Right-click a tag group heading or a tag and select Options > Collapse All.

    2. Click to expand the tag groups you will be using.

  14. Continue with the following procedures to apply tags.
