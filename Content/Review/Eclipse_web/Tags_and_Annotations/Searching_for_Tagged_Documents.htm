Searching for Tagged Documents

To easily find documents tagged (or not tagged) in a certain way:

  1. Log in to Eclipse and open the needed case or batch.

  2. Select the Document Tags or Page Tags tab.

  3. In the selected tab, right-click the tag or tag group title of interest and select one of the following:

    • Search > Search for all tagged documents: Find documents that have been tagged with the selected tag. If a tag group title is selected, documents tagged with any of the tags in the group will be found.

    • Search > Search for all untagged documents: Find documents that have not been tagged with the selected tag. If a tag group title is selected, documents not tagged with any of the tags in the group will be found.

Results display in a new Search Results tab that replaces the contents of the Documents tab.

You can also use "Advanced Search" to find documents tagged in a certain way, using additional search operators.
