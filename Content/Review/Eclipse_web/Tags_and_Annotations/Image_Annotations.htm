Image Annotations

About Annotations

In addition to redactions, you can add the following annotations to images:

  • Mark-ups: Free-form lines

  • Highlights: Transparent color to emphasize particular content.

  • Embedded text (text box): Visible comments added to the image.

  • Sticky notes: Notes indicated with a “notepad” icon; text is visible when the note is double-clicked.

Add a Line or Shape (Mark-up)

If you have sufficient permissions, add a line or shape to an image as follows:

  1. For a selected record, in the Document Details pane’s Image tab, locate the area where you want to draw a line.

  2. Click next to the Annotation button and click the Markup button as shown in this figure.

  3. If needed, select the line characteristics as follows:

    1. Click next to the Markup button.

    2. Select from the options on the Markup toolbar as shown in the following figure.

  4. In the image area, drag in the needed direction to create a line or draw a shape (such as a circle). When finished, release the mouse button.

  5. Repeat to draw other lines or click  or another button to stop the markup activity.

If needed, modify the markup as explained in "Managing Annotations."

Add Highlighting

If you have sufficient permissions, add highlighting to an image as follows:

  1. For a selected record, in the Document Details pane’s Image tab, locate the text or other content to be highlighted.

  2. Click next to the Annotation button and click the Highlight button as shown in this figure.

  3. To change the color of the highlight, click on the Highlight button and select a different color. The color will remain in effect for the current working session or until you change it again.

  4. In the image area, click and drag across content to be highlighted, then release the mouse button.

  5. When finished, click  or another button to stop the highlight activity.

If needed, modify the highlighting as explained in "Managing Annotations."

Add Embedded Text

If you have sufficient permissions, add visible text to an image as follows:

  1. For a selected record, in the Document Details pane’s Image tab, locate the area where you want to add the text.

  2. Click next to the Annotation button and click the Embedded Text button as shown in the following figure.

  3. In the image area, drag to form a box in the area where you want the text to reside.

  4. Complete the following steps to enter needed information in the Embedded Text dialog box. Use the toolbar as shown in the following figure.

    1. Select needed text characteristics (font style, size, and color).

    2. Enter the needed message. You may enter a maximum of approximately 500 characters (including spaces).

    3. To remove entered content, click the Undo button.

    4. If entering a large amount of text, click the Save button while you work.

    5. When finished, click the Save and Close button.

  5. Repeat these steps to add other text messages.

  6. When finished, click or another button to stop the embedded text activity.

If needed, modify the text as explained in "Managing Annotations." For example, if all text is not visible, you may want to resize the text block.

Add a Sticky Note

If you have sufficient permissions, add a sticky note to an image as follows:

  1. For a selected record, in the Document Details pane’s Image tab, locate the area where you want to add the sticky note.

  2. Click  next to the Annotation button and click the Sticky Note button as shown in the following figure.

  3. In the image area, click in the area where you want the sticky note to reside.

  4. Complete the following steps to enter needed information in the Sticky Note dialog box. Use the toolbar as shown in this figure.

    1. Enter the needed message. You may enter a maximum of approximately 500 characters (including spaces).

    2. To remove entered content, click the Undo button.

    3. If entering a large amount of text, click the Save button while you work.

    4. When finished, click the Save and Close button.

  5. Repeat these steps to create other sticky notes.

  6. When finished, click or another button to stop the sticky note activity.

If needed, modify sticky notes as explained in "Managing Annotations" .
