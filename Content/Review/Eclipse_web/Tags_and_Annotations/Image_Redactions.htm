Image Redactions

Working with Redactions

Redactions obscure the content of image files so that only appropriate information is visible in the files you prepare for discovery. In Eclipse, up to 99 redaction categories—with different colors and labels for each—may be used. Redactions can be viewed in Eclipse as solid, outlined, or cross-hatched (if user has appropriate permissions.) The following figure shows a solid redaction in an email image file.

Redact Image Content

If you have sufficient permissions, redact information in an image as follows:

  1. For a selected record, in the Document Details pane’s Image tab, locate the information to be redacted.

  2. Click next to the Redaction button on the Image tab toolbar and select the needed redaction style (Solid, Crosshatch, or Outline). The default redaction style is solid. The style selection applies to all redactions for the current user, until a style change is made. (A change in style will also apply to all redactions for the selected user.)

  3. Note the color on the Redaction button, . If the color (redaction type) is correct, click the button and skip to step 5. If the correct redaction is not selected, go to step 4.

  4. To select the correct redaction, click next to the Redaction button and select the needed redaction from the Redaction Category (top) menu, as shown in this figure.

  5. If needed, click the button again to activate the redaction action.

  6. Drag across the text or image area that is to be redacted. The selected area is highlighted with a dotted line, as shown in the following figure.

  7. When the selection is complete, release the mouse button.

  8. Repeat these steps to apply other redactions.

  9. When finished, click or another button to stop the redaction activity.
