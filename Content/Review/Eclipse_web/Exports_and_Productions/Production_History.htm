Production History

If documents are produced in Eclipse Administration, they will automatically be included in Eclipse (unless your administrator produces the documents without the Update Production Table option).

Viewing Production Documents

To view production documents:

  1. For a selected record, open the Production tab as explained in "Accessing Case Documents." Note:

    1. If only one production document set is available, the production image from that set will display.

    2. If multiple document sets exist, then by default, the first time you access the Production tab, the production image from the first (most recently produced) set available will display. If no image is available in that set, then Eclipse will check subsequent document sets in order by date and display the image from the first set containing a production image for the selected record.

    3. A message will display if no production image is available from any production set.

  2. Optional: Select a different document set from the list in the Production tab toolbar. If you have any questions about the listed document sets, contact your administrator.

    Note: Whichever document set is selected becomes the “working set” for your current Eclipse session. Thus, whenever you select a different record, the corresponding production image from your working set will display unless that set does not contain a production image for the selected record. In that case, the document set list will flash red to indicate the image being displayed is from a different set. (See item b above.)

  3. Scroll as needed to view all of the first page of the document. Continue with the remaining steps as needed to view all content.
