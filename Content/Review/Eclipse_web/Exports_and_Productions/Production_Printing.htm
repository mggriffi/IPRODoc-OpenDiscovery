Production Printing

Printing Production Images

About Printing

Eclipse allows you to print contents of the Production tab. Various printing options exist, such as including a slip sheet. To print a production image, follow the instructions in this section.

Before You Begin

If you have not already done so, read "Production/Export Sets."

Print Production Images

To print production images:

  1. For a selected record, open the Production tab as explained in "Accessing Case Documents."

  2. Select the production set of interest and then click .

  3. In the Print dialog box Options tab, select the needed printer and paper source.

  4. To evaluate the size of the print job, click Calculate. The total number of pages displays to the right of the button.

  5.  A slip sheet can serve as a cover sheet for your print job. To define a slip sheet, click the Slip Sheets tab and select the content to be included. See "Slip Sheets Tab" for details.

  6. To include document content (field data, tag information, or custom messages) in the header and/or footer of the document being printed, click the Endorsements tab and select the content to be included. See "Endorsements tab" for details.

    		7.

    Note:  Production images may include headers and footers defined by your administrator when documents are produced. In this case, endorsements added for your print job may conflict with production endorsements.

  7. When all aspects of the print job are defined, click Print. If a color printer is being used, all colors will be maintained.
