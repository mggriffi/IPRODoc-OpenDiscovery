Use the Concept Wheel

The concept wheel is a visual representation of cluster details, based on the clusters configured by your administrator. It corresponds to the details listed for a selected top-level cluster, as shown in the following figure.

Note:

  • Center of wheel: the cluster group; circle closest to the center shows clusters; fanning away from the center are sub-clusters representing increasing degrees of granularity.

  • The size of each segment is proportional to the number of records corresponding to the cluster details.

  • Color and color depth represent amount and degree of similarity of the data across and among the clusters.

 

The Concept Wheel tab will include data only when a cluster is selected, however.

The Concept Wheel tab can be freestanding (separated from the case pane area). As such, it can be resized/moved within the Eclipse workspace. See Advanced Layout Changes for details on moving components in the Eclipse workspace.

 

Use the Concept Wheel

To use the Concept Wheel:

  1. Complete Evaluate Document Clusters.

  2. Select a cluster in the Analytics list in the Case View pane, then click the Concept Wheel tab in the case pane area.

  3. If the cluster group was selected, the wheel will be centered within the tab. If a cluster or subordinate cluster was selected, the wheel will align to that cluster segment and it will be selected (indicated by a grey color for the segment).

  4. Take any of the following actions:

    • Click a segment to enlarge it and realign the wheel to that area.

    • Point to any wheel segment to view details in a tooltip as shown in the following figure.

    • Drag the wheel.

    • Double-click the wheel to return it to its default position and a size that fits in the existing window.

    • Use the Concept Wheel buttons as described in the following table.
