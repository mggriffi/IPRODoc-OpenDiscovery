Working with Linked Files

A field in Eclipse can be defined as a hyperlink (to a file). Files linked in this way may be any type of file related to a particular record.

To open such files:

  • The field containing the hyperlink must be on a coding form that is available to you.

  • You must have the associated program. For example, if it is a spreadsheet file, you must have Microsoft Excel® or another spreadsheet program.

  • You must have access to the file location.

Open a Linked File

For a selected record, open a linked file as follows:

  1. Log in to Eclipse and open the needed case or batch.

  2. In Record View, select the coding form containing the hyperlink field.

  3. Make sure that editing is not enabled in the coding form.

  4. Click the link.

    5. The file opens in its native program. If your system cannot find the program, you may be asked to find it. Follow the system instructions to locate the needed program.

  5. Read and/or print the file as needed. When finished, close it. If you are asked to save changes, click No.
