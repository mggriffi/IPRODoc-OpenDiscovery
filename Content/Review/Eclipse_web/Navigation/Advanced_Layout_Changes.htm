Advanced Layout Changes

For more advanced layouts of the Eclipse workspace, different regions can be used. Note:

  • Tabs can exist within these regions.

  • The Documents or related tab is treated as a single component with three panes that can be expanded/collapsed (case table, Record View, and Relationships/Tally panes).

  • Most tabs can also be “freestanding,” meaning they can exist anywhere within the Eclipse workspace (within the browser).

  • The Document Details pane can be undocked (freestanding outside of the workspace), including on a separate monitor if you are using dual monitors.

Guidelines for Advanced Layouts

You can best learn what all of the areas and layout options are by experimenting in Eclipse using the procedures explained in this section. The following table explains which components within Eclipse can be moved.

Component/default location

Details

Case View (left slideout pane)

Components:

  • Case Details Dashboard

  • Documents

  • Batches

  • Advanced Search

  • Analytics (optional)

  • Reports

  • Case Administration

  • The Case View pane can be moved or freestanding within the Eclipse workspace.

  • Any item opened from this pane becomes the active tab in the case pane.

  • If the Case View pane is moved, selecting one of the items in the pane may redock it on the left side of the workspace (depends on selection).

Documents (or related) tab (center)

  • Tab corresponding to Case View pane selection

  • Other components depending on what is selected (see details)

  • The active tab in the case pane can be moved or freestanding within the workspace.

  • If the Documents, Batch, Search Results, or Analytics tab is selected, the case pane area includes the Record View and Relationships/Tally panes (whether collapsed or visible). These items cannot be moved separately, but they can be expanded and collapsed.

  • If a cluster has been selected, the Concept Wheel tab displays behind the cluster results. This tab can be moved separately within the Eclipse workspace.

Document Details (right slideout pane)

Components:

  • Image

  • Native Viewer

  • Extracted Text

  • Production

  • Transcripts

  • The Document Details pane can be moved or freestanding within the workspace.

  • This pane can also be freestanding outside of the workspace, including on a separate monitor when dual monitors are used.

  • Individual tabs cannot be moved separately.

Tags (rightmost slideout pane)

Components:

  • Document Tags

  • Page Tags

The Tags pane can be moved or freestanding within the workspace.

Individual tabs cannot be moved separately.

 

Create a Freestanding Tab or Slideout Pane

To make any movable component freestanding, complete the following steps:

  1. If the needed component is collapsed, open and pin it by clicking .

  2. Point to the component’s tab or title bar, then drag.

  3. Continue dragging until the item is in the desired location within the browser area, then release the mouse button.

  4. As needed, resize the freestanding item by clicking and dragging any border.

Arrange the Components

The key to moving tabs to different regions is a set of alignment markers, shown in this figure. These markers indicate the regions in which Eclipse components can reside.

When you drag components in the Eclipse workspace, these markers appear in the center of the workspace and also in each individual area (depending on where the component being dragged is). In addition, selected areas highlight as you position the mouse pointer over the different alignment markers.

The best way for you to learn about workspace layout possibilities is to drag components to different alignment markers. You will see the different areas highlight and become familiar with the options available.

To return to one of your site’s predefined layouts, select it from the main Eclipse toolbar.
