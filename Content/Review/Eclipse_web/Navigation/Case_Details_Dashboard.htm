Case Details Dashboard

The Case Details Dashboard summarizes review progress and provides access to case administrative documents, such as instructions and general (non-discovery) documents. By default, when a case or batch is first opened, this tab displays in the case pane. An example is shown on the next page.

Permissions Required

Access to the Case Details Dashboard and its individual components are governed by specific permissions. Contact your administrator if you have questions about using the Case Details Dashboard.

Evaluate Batch Progress

Progress on batch reviews for an open case is summarized on the Case Details Dashboard as follows:

  • Eclipse administrators can see progress on all batches for a case, for a review pass, or an individual user.

  • For cases they administrate, Case Administrators can see progress on all batches for a case, for a review pass, or an individual user.

  • Individual users see their progress on all batches assigned to them.

Use the Case Details Progress Chart

To evaluate a case's batch progress:

  1. Log in to Eclipse and open the needed case or batch.

  2. If the Case Details Dashboard is not the active tab in the case pane, click Case Details Dashboard in the Case View pane.

  3. In the Case Details Dashboard menu, select the scope of the review:

    • Click Batch Review Status to see that status of all batches of the case.

    • Click a specific review pass in that list for corresponding status.

    • Click a batch for its corresponding status.

    • Administrators only: click a user for his/her status. This figure shows the details an administrator sees. A user would see the batches to which they are assigned.

  4. In the Case Details Dashboard, review the chart.

  5. As needed:

    • Change the display type, Doughnut, Bar, or Pie Chart.

    • Select chart criteria, Document or Batch Status.

    • Enlarge the Eclipse workspace area for a better view (for example, maximize the browser size or unpin slideout panes).

    • Pie charts: If desired, point to or click individual sections to better evaluate specific details, as shown in this figure.
