Eclipse FAQs

Can I print several PDFs at once?

Yes, batch printing is available.

Can I split PDFs in Eclipse?

The current version does not support document unitization. has additional tools that are available if this feature is important to you.

Will your document viewer show video?

Currently the viewer does not support video. Coming soon is our HTML5 version which will showcase synchronized transcripts and video.

How do you handle video in Eclipse?

Eclipse allows you to easily download/view the native file through the software depending on your native rights. As long as you have the native application loaded on your local computer, you will be able to view the document locally.

I cannot search for the BEGDOC number. Why?

The BEGDOC field must be included within the index for the case from within the Eclipse Admin module, please consult the case administrator for details.

How do I remove the timeline from my review space for more of my screen real estate?

Click the undock/unpin button found in the upper right.

How do I activate a batch of document review that I was assigned?

After a successful case login, click the Batches Expander along the left side of your screen, and click ‘Get Next Available Batch’ and select the review pass assignment to receive a batch of documents. The batch will show as ACTIVE onscreen and you can now review within the Documents expander.

I have an ACTIVE batch for document review but I don't see them yet, only ALL DOCUMENTS within the case?

Check the Documents Grid area and select ‘My Batch’ from the drop-down menu to see your Active batch of documents for review.

Can I do manual coding in Eclipse?

Review allows the creation of coding fields/forms for reviewers to complete coding projects.

How do I add IPRO into my Trusted Sites?

In addition to having an approved version of Internet Explorer, before you access Eclipse the first time, add the Eclipse site to Internet Explorer’s list of trusted sites:

  1. Obtain the Eclipse URL. In Internet Explorer, select Tools > Internet Options.
  2. Click the Security tab, then Trusted sites.
  3. Click Sites as shown in the adjacent figure.
  4. In the Trusted sites dialog box, A) enter the Eclipse Website’s URL, B) optional: clear the verification option (if the protocol is not HTTPS), and C) click Add.

Can I filter the grid display?

The document grid can be filtered by column to allow for quick and easy retrieval of specific values. Each column can have one filter. Filters that are applied across two or more different columns use an “and” operator when the filter is applied. Users can also defer the wait time for filtering the grid when multiple columns are used by building a filter. Building a filter delays submitting and reloading the document grid until all of the filter values are chosen. When a “contains” operator is needed, use the * (wildcard) before, after, or on both sides of the filter value.

How do I reset my screen layout?

Click the Case Name drop down menu found in the upper left to reset the default layout or other defined screen layout.
