Work With Batches

Individual batches are assigned to individual reviewers. They allow you to efficiently divide the work, assign it to the correct individuals, and monitor progress. Individual batches are created as part of Review Pass creation.

Check In Batches

Checking in a batch is available through the Review Grid layout. When entering the review pass for the first time, the Document Viewer window automatically opens in the foreground and displays the first document in the batch. The review grid remains in the background. The grid displays a highlight for the corresponding document shown in the Document Viewer window.

Proceed to review the documents in the batch. When ready, select the Check In button above the review grid.

To check in a reviewed batch, complete the following steps:

  1. From the Document Viewer window, follow Conduct ReviewConduct Review for each document in the batch.
  2. After the last document is reviewed for the batch, close the Document Viewer window.

  3. From the review grid, click to display the Check In Batch dialog box as shown here:

  4. The option Open next available batch is checked by default. Uncheck the box if you do not want to review the next available batch.

  5. Click Submit. A prompt displays to indicate that the batch was checked in, and the Document Viewer window opens displaying the first document for the next batch provided that the option, “Open next available batch”, was selected.

    Note: If the Open next available batch option was not selected, the main Review home page appears.

Reassign a Batch

An incomplete batch may be submitted by the batch reviewer. This batch can be accessed by another reviewer. The system automatically un-assigns a batch after 30 minutes of inactivity by default. The Case Batch Timeout can be modified through Batch Management in the review case. Reassigning a batch is also available through Batch Management. For more information, see Manage Batches.

When entering the review pass for the first time, the Document Viewer window automatically opens in the foreground and displays the first document in the batch. The review grid remains in the background. The grid displays a highlight for the corresponding document shown in the Document Viewer window.

To un-assign a partially reviewed batch, complete the following steps:

  1. With remaining documents in the batch, close the Document Viewer window.

  2. From the Review grid, click to display the Check in Batch dialog box as shown here:

  3. Clear the option Open next available batch if you do not want to review the next available batch. This option is selected by default.

  4. Click Submit. A prompt displays that indicates the batch was checked in, and the Document Viewer window opens to display the first document for the next batch provided the option, Open next available batch, was selected.

    Note: If the Open next available batch option was not selected, the main Review home page appears.
