Create Review Passes and Batches

Review passes are based on searches saved in . There are two modes of review passes: disabled and enabled. With disabled review passes, searches are planned and saved in before you create the review passes. With enabled review passes, you must scope the documents by using either Entire Case, Saved Search, or Custom Index. Coding forms are also required for a review pass. For a detailed overview, see Review Passes and Batches.

Note: disabled review passes still run “in the background”, but it is transparent to reviewers. At some point in the review, the setting could be enabled ( enabled), to start prioritizing predicted positive documents within the review pass, and that would change the document order for reviewers.

If you prefer to turn off the functionality running “in the background” on disabled review passes, the functionality can be turned off across the environment through SQL. For more information, please contact Support.

Create Review Passes

You can create either an enabled review pass or an disabled review pass.

What is Active Learning?

Active learning is the ability to have documents prioritized through the use of algorithms with no direction from the user beyond the coding supplied when they review the documents. With , the prioritized documents are more likely to be relevant to the review purpose selected.

The “continuous” facet of relates to the process of ranking and re-ranking documents for manual review based on the constant stream of incoming coding throughout the review lifecycle. That is, reflects not only an initial set of assessments to rank and prioritize documents, but continuously updates those rankings based on the most recent assessments.

Review passes are designed for to be enabled or disabled. When creating a review pass, -enabled is the default.

Create Batches

To create batches:

  1. In the Review Pass Management panel, click the required review pass. If a large number of review passes exists, take any of the following actions to find the required one:

    1. Scroll vertically to locate the review pass in the list.

    2. Sort the keyword list names by clicking the sort button, , and selecting from the different options. Click a selected option again to reverse the sort order.

    3. In the Search box , start typing any of the following items: the keyword list name, the name of the administrator who created/modified the group, or part of the created/modified date. This is a plain text search that searches these items in order.

  2. Click Update Documents and Create Batches if it is required for unbatched documents to be batched. For details, see Update Batches.

  3. Review the number of unbatched documents in the review pass, then click Create Batches.

  4. Repeat the previous steps to create other review passes.

Update Documents

To batch unbatched documents for the selected review pass:

  1. Click Update Documents and Create Batches. The following dialog box appears:

    Click Continue. A message displays to let you know the updated batch is being built.

  2. View the Batch Information section to see the updated batch status.

Re-batch Unreviewed Documents

To re-batch unreviewed documents:

  1. If it is required that you create a new review pass, first complete the Create Batches section.
  2. Click Re-Batch Unreviewed Docs.
  3. After it is determined in the Review Pass Information area that there are unreviewed documents, click Re-Batch Unreviewed Docs to have these documents re-batched into new batches.
