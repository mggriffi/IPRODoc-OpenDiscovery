Edit and Delete Review Passes

Modify or delete review passes as required by following the steps provided.

Edit a Review Pass

After a review pass is defined, you can change certain parts of its definition.

Note: If batches have already been created, some changes will have no effect on existing batches (for example changing batch size). Changing the search on which the review pass is based will add documents that were not results of the first search to the review pass.

To change a review pass definition:

  1. Open the review pass you would like to modify. To view the parameters that have been defined for the review pass, open the Settings tab. See Settings Tab for more information about each parameter.

  2. To edit your review pass, select the Edit icon, , in the top-right corner.

  3. Adjust settings as needed, and select Save.

  4. Inform users of changes.

Delete a Review Pass

To delete a review pass (all batches will also be deleted):

  1. Select the relevant review pass from the list.

  2. For the selected review pass, click ; in the confirmation dialog box, click OK.
