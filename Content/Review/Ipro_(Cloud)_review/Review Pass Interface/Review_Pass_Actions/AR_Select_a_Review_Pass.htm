Select a Review Pass

This topic explains how to select a particular Review pass within a case.

  1. Click ECA & Review on the Home page or from the main menu.

  2. As explained in About the Review Home Page, on the Review home page, click the desired case card. The review passes load in the funnel graph.

  3. Select the desired review pass located on the right side of the funnel graph. The Review pane opens with the batch document set, pre-defined relationships, and the coding form/tag palette.
