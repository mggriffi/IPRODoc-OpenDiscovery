Overview: Document Viewer

The Document Viewer provides an interface for viewing document properties as well as manual tagging and editing fields for individual documents.

For an overview of the document viewer, click the video below.

The Document Viewer consists of the following:

  • Relationship Menus: Displays the default Template relationship menus (Duplicates, Family, Near Duplicates, Email Thread, Similar Documents and Document History). See Overview: Relationships for more information.

  • Viewer Option Icons: Display the document in a separate Document Viewer and split the Document Viewer screens.

  • Document Viewer tabs:

    • Web Viewer: Displays natives.
    • Image: Displays TIFF, JPG, PNG, or PDF files.
    • Production: Displays the production images.
    • Text: Displays extracted text field for each document.
    • Metadata: Displays the metadata fields and corresponding values for the document.

  • Coding Form Pane: Displays fields associated with the current coding form, and the associated tag palette.

    Note: If you are working in a Review Pass, the Review Status will also appear under the selection window. Options include Not Reviewed (default), Reviewed, and On Hold.
  • Tag Palette Pane: Displays the tags associated with the selected coding form.

  • Review Navigation toolbar: Save changes to the current document, move to the next document for review, or undo changes.
