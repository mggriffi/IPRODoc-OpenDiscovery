Right-Click Tag Options in the Document Viewer

Add a Tag to a Tag Group

This function is also explained in educational content on the IPRO Learning Center. Click the following link to be redirected to the course: On a Mission: Creating Tags on the Fly in

In the Document Viewer, you can simply and quickly add a new tag to a particular tag group by doing the following:

  1. In , click a case card.

  2. Click the Enter Case button.

    The Visual Search dashboard appears.

  3. In the Search Results panel at the top right of the dashboard, click a link to a document in the BEGDOC column. The Document Viewer appears.

  4. In the tag pane to the right, right-click a tag group in which you want to add a tag name.

  5. Click Add Tag, type the tag name, and click .

Rename a Tag

In the Document Viewer, you can simply and quickly rename a tag by doing the following:

  1. Follow the steps in Add a Tag to a Tag Group.
  2. In the tag pane to the right, right-click a tag that you want to rename.

  3. Click Rename Tag, type the tag name, and click .

Search for Tagged/Untagged Documents

In the Document Viewer, you have the ability to create a search to quickly find all documents where a specific tag has been applied, or to find all documents that do not have that tag applied. The search criteria is added to the queue in the Visual Search dashboard and can be run as is, or updated with additional search criteria.

For an overview of adding a tag to search criteria, click the video below.

  1. Follow the steps in Add a Tag to a Tag Group.

  2. In the tag pane to the right, right-click the tag you want to search.

  3. Select Add Tag to search criteria.

  4. The selection window opens giving you the choice to search for all tagged documents or all untagged documents.

    Note: The tag must be selected to create the search.

  5. Once a selection is made, it will be added to the search queue in the Visual Search dashboard.

  6. Click to run the search.
