Family

Intended to show documents that are related to one another, as in an email and its attachment. Family relationships are based on the content of the default BEGATT field and sorted by the BEGDOC field. These column fields display by default.

The Family panel is created for each Case by default and displays the parent and its children in a list format. The current document being reviewed is highlighted in the list. The grid shows the file type represented by an icon. If the file type cannot be determined through its metadata, a broken image icon appears.
