Manage Mass Actions in the Relationship Panel

Mass Actions applied from the Relationship Panel in the Document Viewer Window allow you to tag multiple documents at the same time based on document relationships. See Overview: Relationships for more information.

Apply Mass Actions from the Document Viewer Window

  1. Open the Document Viewer Window by clicking on a needed document from the Results Grid.

  2. From the Relationship pane, expand the desired relationship menu and pin it in place by clicking .

  3. Select the check box to the left of the field columns to display the button as shown in the following figure. All items are selected.

  4. Do any of the following:

    1. Clear items to be excluded for mass actions.

    2. Clear the check box to the left of the field columns to clear all items in the menu.

    3. Select individual items for mass actions.

  5. Click to display the coding form/tag palette dialog as shown in the following figure.

    Note: Coding form rules are not enforced when tags are applied in this manner.

  6. Add individual tags by moving the slider over to the right as shown here:

  7. Remove tags by moving the slider to the left as shown here:

    Note: When the slider control is left in the center, the tag remains unchanged as shown here:

  8. Click Apply at the bottom of the window. The Mass Action dialog closes and applies the fields/tags to the selected documents. The previously selected documents in the selected Relationship menu revert to the default status; they are all cleared. The coding form/tag palette reflects the changes made in the Mass Actions dialog.

    Note: A prompt may appear if invalid characters and/or formats are entered in the coding form or if the number of characters exceed the amount allowed for a file

  9. Repeat the previous steps for other groups of documents needing to be tagged in the same way.
