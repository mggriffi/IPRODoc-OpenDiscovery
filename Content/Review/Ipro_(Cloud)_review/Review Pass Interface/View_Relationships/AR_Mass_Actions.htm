Apply Mass Actions

Mass Actions are actions performed on multiple documents at one time such as populating coding fields and/or applying tags. Existing permissions apply for mass actions. Mass actions are performed from the Search Results tab or the Document Viewer Window.

  • Mass Actions applied from the Search Results tab allow you to OCR, Print, Image, tag, and perform other actions to multiple documents at the same time.
  • Mass Actions applied from the Document Views Window allow you to tag multiple documents at the same time based on document relationships. See About Relationships for more information.

To learn more about applying mass actions from both locations, read through the sections below.

Apply Mass Actions from the Search Results Tab

  1. Log into as an administrator.

  2. Complete the initial steps to open the Search Results tab or the Review Grid.

  3. Optional: Sort the documents in the Search Results tab or the Review Grid. (See Work with Grid Columns for details on sorting, filtering, etc.)

  4. All documents: To perform a mass action on all documents in the current document set:

    1. Click the check box in the column heading row to select all the documents as shown in the following figure.

    2. Skip to step 6.

  5. Selected documents: To perform a mass action on selected documents in the current document set:

    1. Select the check boxes associated with any documents you would like to perform a mass action as shown in the following figure.

    2. Continue with the next step.

  6. Select in the mass action toolbar, then click the mass action you would like to perform. Refer to the image below.

  7. Complete one of or more of the following mass action procedures.

    • Tag Multiple Documents

    • Edit Multiple Fields

    • Export Data for Multiple Records

    • Assign Documents to a Review Pass

    • Perform Mass Action Print to PDF
    • Perform Mass Action Imaging
    • Perform Mass Action OCR

  8. Repeat this procedure for other groups of documents that need mass actions performed the same way.

Apply Mass Actions from the Document Views Window

  1. Open the Document Views Window (also known as the Review Pane) by clicking on a needed document from the Search Results tab.

  2. From the Review pane, expand the desired relationship menu and pin it in place by clicking .

  3. Select the check box to the left of the field columns to display the button as shown in the following figure. All items are selected.

  4. Do any of the following:

    • Clear items to be excluded for mass actions.

    • Clear the check box to the left of the field columns to clear all items in the menu.

    • Select individual items for mass actions.

  5. Click to display the coding form/tag palette dialog as shown in the following figure.

    6. Note: Coding form rules are not enforced when tags are applied in this manner.

  6. Do any of the following:

    • Modify the coding form fields as needed.

    • Select the applicable tag group.

    • Add individual tags by moving the slider over to the right as shown in here:

    • Remove tags by moving the slider to the left as shown in here:

    Note: When the slider control is left in the center, the tag remains unchanged as shown here:

  7. Click Apply at the bottom of the window. The Mass Action dialog closes and applies the fields/tags to the selected documents. The previously selected documents in the selected Relationship menu revert to the default status; they are all cleared. The coding form/tag palette reflects the changes made in the Mass Actions dialog.

    8. Note: A prompt may appear if invalid characters and/or formats are entered in the coding form or if the number of characters exceed the amount allowed for a field.

  8. Repeat the previous steps for other groups of documents needing to be tagged in the same way.
