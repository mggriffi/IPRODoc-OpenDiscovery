Filter Search Results

Once your search has been run the results can be filtered to further define what you are searching for. This can be accomplished through the results grid filtering or by quick searches within the results.

Filter by Column Data

To filter search results based on column data:

  1. In the heading of a column you want to filter, enter the term in the filter box.

  2. Click Enter to run the search. The filter search will appear above the grid.

    Note:

    • This search is not case sensitive.

    • This search uses the CONTAINS operator.

    • Wildcard support: An asterisk (*) can be used to represent zero or more characters.

  3. To filter by one or more additional columns:

    • Select your column and add the term to the filter box.

    • Click Enter to run the search. The additional filter search will appear above the grid.

    Note: Filters on different columns are treated as AND statements.

  4. When finished, the results will be displayed. The number of filtered search results will be displayed at the bottom of the page in Visual Search.

    Alternatively, the results are displayed on the Results Tab in Advanced Search.

Note: The results from filtering can not be saved as is. The original search must be modified to include the filters, or the results must be tagged and a new search created.

Remove Filters

To remove a filter, click next to the search above the column header row, or the button on the far right.

Filter by Quick Search

The Quick Search bar is located next to Actions and is titled "Filter Search Results".

 

 

Note: If a search relationship has been selected in a Visual Search or Advanced Search, that search relationship displays in the Quick Search bar and can't be altered for your quick search.

If a search relationship has not been selected in a Visual Search or Advanced Search, the Relationship menu is available. When you select a relationship from the menu it will modify rather than filter the original search. For more information on search relationships, see Overview: Relationships.

  1. In the Filter Search Results box, type the text string to search for and either click or press Enter.

    Note: Only one text search can be run at a time. If you enter another text term, the first quick search will be replaced.

  2. View your filtered quick search results. The results will be displayed at the bottom of the page in Visual Search.

    Alternatively, the results are displayed on the Results Tab in Advanced Search.

    Note: The results from quick searches can not be saved as is. The original search must be modified to include the quick search terms, or the results must be tagged and a new search created.

  3. To clear your quick search, see Remove Filters.
