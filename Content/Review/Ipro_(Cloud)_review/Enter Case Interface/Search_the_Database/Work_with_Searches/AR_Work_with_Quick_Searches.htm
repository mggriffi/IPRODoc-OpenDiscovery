Work with Quick Searches

Visual Search and Advanced Search Quick Searches

To run and/or save a Quick Search in the Visual Search tab or the Advanced Search tab:

  1. After reviewing your search definition in the Visual Search tab or Advanced Search tab, complete either step 2 or step 3.

  2. To run the search without saving it, click .

  3. To save the search:

    1. Click .

    2. Enter a unique name in the Search Name field.

    3. Select the folder in which the search should be stored. If the folder does not exist, create it as explained in Work with Public, Shared, and Private Search Folders.

    4. Click to save and run your search.

  4. To save the search without running it, click .

  5. If you run the search, evaluate the outcome in the Results tab.
  6. To view details for a specific document, click the document link. The Document Viewer opens.

    7. You can view the image by clicking different tabs of the Document Viewer, Web Viewer, Production (if available), Image, Text or Metadata tabs.

  7. Note: If no results exist for the tab, the tab will be blank.

Search Results Quick Searches

To run and/or save a Quick Search on resultant documents on the Results tab:

  1. On the Results tab, select the resultant documents to include in your search. Then perform either step 2 or step 3.

  2. If a search relationship has not been selected in a Visual Search or Advanced Search, the [Include Relationship] drop-down menu is available. Select the desired search relationship, for example, "Family."

  3. If a search relationship has been selected in a Visual Search or Advanced Search, that search relationship displays in the Quick Search bar. You can use only that relationship for the Quick Search.

  4. In the search text box, type the text string to search for and either click or press Enter.

    5. When the search completes, the total number of documents found is displayed in the Results tab, the search criteria is shown, and the documents are listed.
