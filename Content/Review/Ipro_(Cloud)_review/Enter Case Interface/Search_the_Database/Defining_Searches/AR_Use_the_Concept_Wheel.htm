Use the Concept Wheel in

The concept wheel is a visual representation of cluster details, based on the clusters configured by your administrator. It corresponds to the details listed for a selected top-level cluster, shown as follows:

Note:

  • Center of wheel: the cluster group; circle closest to the center shows clusters. Fanning away from the center are sub-clusters that represent increasing degrees of granularity.

  • The size of each segment is proportional to the number of records corresponding to the cluster details.

  • Color and color depth represent amount and degree of similarity of the data across and among the clusters.

For more information, see Define a Visual Search.

Using the Concept Wheel

To use the Concept Wheel:

  1. In the Case View, select the Visual Search tab and then click .

  2. On the Concept Wheel, click a cluster group, a cluster, or a subordinate cluster.

  3. If the cluster group is selected, the wheel will align around the group. If a cluster or subordinate cluster was selected, the wheel will align to that cluster segment or subordinate cluster segment.

  4. Take any of the following actions:

    • Point to any wheel segment to view details in a tool tip as shown in the following .

    • Click a segment to enlarge it and realign the wheel to that area. Note that the more granular the concept becomes, the darker is the hue.

    • Click a wheel segment and it centers in the segment.

    • Click the center segment until the wheel returns to the default position. The number if clicks required to return to the default position depends on how granular the concept is.

    Tip: The bar above the wheel shows concept granularity and can be used to return the wheel to a less granular state. By clicking All Documents, you can return the wheel to the default position.

    Other ways to return the wheel to the default position:

    • Drag the wheel.

    • Double-click outside the wheel.
