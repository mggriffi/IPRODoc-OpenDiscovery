Select Numbering Options

This area of the form allows you to define the way in which you want exported documents numbered.

To select numbering options:

  1. In the export set form, click Numbering Options if this area of the form is not visible.

  2. Complete this section as described in the following table.

    3. Bates Numbering

    Option

    Description

    Continue Previous Numbering

    Criteria for use

    If the following conditions are true, complete the following procedure.

    • Multiple export sets comprise a single export job and you want continuous numbering for the entire job.

    • You are creating new numbers, not using original document numbers.

    • The export set being configured is not the first set in the complete job and a previous export set has been completed.

    Procedure

    1. Click Continue Previous Numbering.

    2. In the resulting dialog, select the export set that numbering should follow and click OK.

    3. Skip to skip to step 3.

    Note: Choosing to continue previous numbering and then creating a new numbering scheme negates the Continue Previous Numbering setting.

    Document Numbering

    Select the needed option:

    • Create new numbers: Re-number pages/documents using the remaining options in this section of the form.

    • Use image priority: Select one or more document sets from which documents will be selected. See the following paragraphs.

    Image priority details

    Basic information:

    • Document sets include original images and production sets that have previously been created.

    • Each document will be numbered as it was in the set from which it is taken.

    • In addition to selecting the document sets, use the Up and Down buttons to organize the sets. The following figure shows an example in which multiple document sets have been selected and organized.

    When multiple sets are selected, documents will be chosen as follows:

    • looks first in the image set at the top of the list; if the image is found, it will be produced.

    • If the image is not found in the first set, checks the second set; if the image is found, that is the version that will be produced.

    • If the image is not found in the first/second sets, continues to check remaining image sets in the Selected list, from top to bottom.

    • As soon as it finds the image, that is the version that will be produced.

    • If no image is found in any set, a conflict will be noted during validation as described in Export/Production Troubleshooting.

    Type

    If you are creating document numbers for the export set, select a numbering scheme:

    Basic: Includes text prefix, delimiter, starting page, and page number width options. Each page will be numbered as a separate document.

    Advanced: Includes basic options plus the ability to define document numbering in addition to page numbering. Each page will be numbered using document and page identification.

    3 level rollover numbering: Includes prefix, starting parent, parent width, document delimiter, starting document, document width, page delimiter, starting page, and page width. This option applies for the Ringtail load file format and Output location.

    Basic Options - define for basic and advanced numbering.

    Prefix

    Optional. Enter a prefix and optionally select a delimiter for your new numbering scheme.

    Prefix must be alphanumeric prefix (no special characters).

    Prefix Delimiter

    Starting Page

    Enter the starting page number (a whole number greater than zero).

    Starting Parent

    Starting parent field has default value of 1. Enter a numeric value only.

    Parent Width

    The default numeric value is 8. Use the spin controls to increase (maximum 9) or decrease (minimum 1) the numeric value or type a numeric value directly in the field. This value should be greater than or equal to the width of the starting parent number.

    Document Delimiter

    Select a delimiter to be used between the Starting Folder and Starting Document. The default setting is None.

    Page Width

    Enter the number of digits to be used for page numbers (greater than or equal to the size of the starting page number). Leading zeros will be used as needed.

    Example

    This example matches the numbering scheme you have defined. Review it to make sure the scheme is correct.

    Advanced Options - select all of the above options, plus the following.

    Starting Document

    Enter a starting document number. The default starting value is 1.

    Document number must be a whole number greater than zero.

    Document Width

    Enter the number of digits to be used for document numbers (greater than or equal to the size of the starting page number). Leading zeros will be used as needed.

    Page Delimiter

    Select a delimiter to be used between the document and page numbers. The default setting is None.

  3. If the export set being defined is not for formal production, skip to step 5. For a production set, go to step 4.

  4. The selections in the following table require user-defined FixedLengthText fields that correspond to the data described for each field. In addition, security for these fields must allow updates by the user creating this production set. Data for the current production will overwrite data in the fields.

    5. Production Numbering Fields

    Option

    Description

    Begin Bates

    Requires fields for storing beginning- and/or ending-production numbers. Select these fields.

    End Bates

    Begin Attachment Bates

    Requires fields for storing beginning- and/or ending-production numbers for attachments. Select these fields.

    End Attachment Bates

    Custom Text Field

    If your case includes a field for storing production-related notes, select the field in which the notes should be stored.

    Custom Text

    If you select a custom text field, enter notes for the production in this field.

    Note: Production Numbering options are visible to users only if they have the Production permission.

  5. When all numbering options have been defined, either complete other procedures in this section or click Save to save the current definition.

  6. When the definition is complete, go to Complete the Export.
