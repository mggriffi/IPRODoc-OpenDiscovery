Search for Keywords

You can search for key terms and sort the summary columns to quickly find your results.

  1. Open Keyword Management and select your keyword list. See Open Keyword Management.

  2. On the Summary tab, enter your search term below Keywords. The results will automatically display.

  3. Click the to clear your search.

  4. The summary columns can be sorted by clicking the arrows within each heading.

Add New Keywords

New search terms can be added to an existing keyword list at any time.

  1. Open Keyword Management and select your keyword list. See Open Keyword Management.

  2. Click Add Keywords.

  3. Add your new keyword terms or phrase and click Save.

  4. Anytime a keyword list is changed by adding new keywords the job status will change to Out of Date and must be reprocessed to include the new terms.

  5. Click Process Now to set the reprocessing job in motion. The job will run based on your original Time Frequency setting.

Changing a Saved Search Scope

If the keyword list was created using a saved search for the scope, any changes to the saved search will require an update to the processing of the keyword list. You will see the following:

  1. The Status will change to Out of Date.

  2. It will be indicated in the Keyword List Details when the search was modified.

  3. Click to set the reprocessing job in motion. The job will run based on your original Time Frequency setting.

Note: The progress of all reprocessing of the Keyword List can be monitored in the Job Manager by clicking the View Job link.

Edit or Delete Keyword Lists

You can edit the keyword list details or delete the list if needed.

  1. Open Keyword Management and select your keyword list. See Open Keyword Management.

  2. Click the Edit or Delete icons to the right of Keyword List Details.

    • Edit: Opens the Edit Keyword List Details window.

      Note: The Keyword List Name and the Search Scope cannot be changed once created.

    • Delete: Permanently deletes the Keyword List.

      A message appears confirming the deletion.

      Note: Keyword tag groups, tags, and fields will remain with tags applied and field content based on the last time keyword results were saved.
