Overview: Keyword Management

In Keyword Management you have the ability to define lists of related search terms (words, phrases, or expressions) of particular importance, and find documents containing these terms. Keyword searching is a basic text search of all indexed fields.

Keyword searching can assist you with document management, review, and other tasks. For example, keyword searching can be used to identify the documents needed for a specific review.
