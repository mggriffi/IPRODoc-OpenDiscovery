Assign Keyboard Shortcuts to Tags

allows you to assign keyboard shortcuts to tags in order to further streamline the process of reviewing documents. After a keyboard shortcut has been assigned, users can simply press the keys associated with that shortcut to apply the tag to a document. Keyboard shortcuts are managed on the Tag Palette tab, under Case Settings in the Review module. For instructions about how to review documents by using keyboard shortcuts, see Use Keyboard Shortcuts To Review Documents.

To assign a keyboard shortcut to a tag:

  1. In , click a case card.

  2. Click the Enter Case button.

    The Visual Search dashboard appears.

  3. On the left side of the Case View page, click the Case Settings icon. The case settings tabs display in a ribbon bar at the top of the screen.

  4. Click the Tag Palette tab. The Tag Palette screen displays.

  5. Select a pre-defined tag group from the left-panel that contains the tags to which you want to assign shortcuts.

    Note: If you want to sort the tag palettes list, click to display the sort menu and select an option.

  6. The selected tag group appears, along with its associated tags. Select the Edit icon above the tag group to make changes.

  7. Define shortcut keys for any tags in the group. By default, the shortcut combination consists of the SHIFT button plus a key of your choosing. Select the key from a drop-down menu. Keys that have been applied to other tags display as grayed out and cannot be selected.

  8. Follow the steps to assign shortcut keys to other tags in the tag group currently being modified. When finished, click Save.
  9. Proceed to assign shortcut keys to tags in other tag groups, following the above step.
  10. Inform reviewers of the new keyboard shortcuts available.
