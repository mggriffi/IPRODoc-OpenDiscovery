Manage Pick Lists

A pick list is a set of items (terms) from which a reviewer can choose when editing a field in Record View. The following figure shows an example of a pick list field CODING DOC TYPE. Clicking the down arrow opens the pick list items.

Because the set of items in a pick list may be appropriate for more than one field in a case, the definition of pick lists is separate from the definition of the case’s fields.

If a field is modified to be a pick list after data is imported, all data for the field will remain intact. Reviewers needing to revise the field will select an item from the pick list.

Plan Pick Lists

As with all tasks, proper planning will help ensure that the pick lists you create are useful to your reviewers. For each pick list, consider such factors as:

  • Can the pick list be designed to apply across multiple fields (such as a field for which Yes and No are needed entries) or is it specific to only one or a few fields?

  • Have all needed choices been identified?

  • What kind of naming/terminology will be most useful to reviewers? (For example, for names, should first and last names be included or just last names?)

  • Pick lists items are listed in alphabetical order. If one item will be selected the majority of the time, consider preceding its name with a special character (such as an asterisk or underscore) to keep it at the top of the list.

Create Pick Lists

To create a pick list:

  1. Review Manage Review Case Settings for instructions on opening the case settings options.

  2. Click the Pick List tab.

  3. In the Pick Lists panel on the left side, click . The fields appear on the right.

  4. In the Name field, enter a name for the list.

  5. In the Terms field, enter items to comprise the list, one item per line. Maximum size for each term is 1,000 characters.

  6. When finished, click Save.

  7. Repeat this procedure to create other pick lists.

  8. Explain pick list details to reviewers who will be editing fields in .

Modify Pick Lists

Complete the following steps to modify or delete a pick list through case settings.

  1. Review Manage Review Case Settings for instructions on opening the case settings options.

  2. Click the Pick List tab.

  3. Select the pick list to be modified from the pick lists on the left side.

    Note: If you wish to sort the pick lists, click to display the sort menu and choose an option.
  4. Click .

  5. Review the terms in the Terms list (on the right) and modify the terms as needed.

  6. When finished, click Save.

  7. To delete a pick list, select the pick list on the left side and click . Click OK in response to the confirmation message.

  8. Notify users of the changes that were made.
