Create and Manage Content Filters

Content filters are those repetitive phrases, addresses, or words that are not relevant to your case. For example, confidentiality notices, email signatures, company names, etc. This topic describes how to create and manage Content Filters that are used to determine what information should not be included in the analytic index for your case.

Create Filters

  1. Start Eclipse Admin and log in.

  2. On the Case Management ribbon, select Eclipse Analytics > IPRO Analytics Content Filters.

  3. Select the needed managing client and case.

  4. There are two options to add content filters. When you click , the filter terms will be generated automatically.

    Note: When you select "Auto -generate" the system will create multiple filters. Depending on your data, this sometimes extracts unusual characters that can cause processing errors. It is recommended that you review the content filters created. You can edit or delete unwanted filters.

  5. The second option is to create your own filters. Click to add customized text filters to your list. This will open the Content Filter Text dialog window.

  6. After adding your text, click OK.

Edit a Filter

  1. To edit a filter, click the corresponding button next to the filter you would like to change. This will open the Content Filter Text dialog window.

  2. Make any necessary changes to the text in the filter and click OK.

Delete Filters

  1. To delete a filter, click the corresponding button next to the filter you would like to delete. This will open a dialogue box that confirms whether or not you “want to delete all filters.” This action is permanent, so double-check to make sure you have selected the correct filter before continuing. If you would like to proceed deleting the selected filter, click OK. If not, click Cancel.

    Note: If you would like to delete ALL of the filters associated with a selected case, click the Delete All button and follow the instructions in the dialogue box.
