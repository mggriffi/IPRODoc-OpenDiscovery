Create Email Full Thread Search

For email threading, one way of reviewing batches is using an email full thread search. In this scenario, you create batches such that a single reviewer can review all documents in every thread by grouping all emails of a thread into the same batch. Allowing for all documents in the same email thread to be reviewed by one reviewer provides for faster, more consistent review decisions.

  1. Open the Advanced Search window to create a new Field Search. Include Families on the search by ensuring Family is selected from the drop-down.

  2. The criteria of the search should identify all families that received an Email Thread value. This can be accomplished by searching for any documents where the IA Email Set ID field contains a value. For more information about the IA Email Set ID field and other email threading fields, see Understand Email Threading Fields.

  3. Click and provide a name for the search. For example you could name the search Email Thread Full Batch.

  4. When creating a new Active Learning Disabled Review Pass use your Advanced Search for the documents to be batched. See Manage Review Passes for additional information.

    Note: When creating an Email Full Thread review batch the following fields should be selected:

    • Set the family field to IA Email Set ID to ensure all families and threads are kept together.

    • Set the Batching Order to IA Email Sort/Descending.
