Get Started - Manage Review

The Review module of allows users to review, analyze, and annotate case documents with efficient, easy-to-use tools. Administrators working in Review can complete various tasks to manage their Review cases, including:

  • Use advanced and/or visual search to efficiently find case data

  • Create and manage review passes

  • Quickly identify email threads for more efficient review

  • Schedule keyword searches

  • Create productions and exports

For more information on case setup and other tasks typically done by a Case Administrator, see Review: Manage Your Review.

For more information on conducting a review, see Review: Conduct Your Review.
