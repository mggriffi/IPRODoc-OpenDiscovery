Tag Multiple Documents

If you have the permissions needed to do so, use the following procedure to tag multiple documents in the same way all at once.

Note: For this procedure, you do not have to enable tagging in the Record View pane.

  1. In the Documents or related tab’s toolbar, click actions to open the mass actions toolbar.

  2. Select Edit in the mass actions toolbar and complete step 3 or 4 as needed to select the documents to be tagged.

  3. All documents: To tag all documents in the current document set (based on your selection during preparation):

    Note that all documents in a batch or an entire case may be tagged, although such selections may be too large to be processed effectively.

    1. Select All, from the mass actions toolbar as shown in the following figure.

    2. Skip to step 5.

  4. Selected documents: To tag selected documents in the active tab (Documents, Batch, etc.):

    1. Select the options on the left side of the needed documents in the case table as shown in the following figure.

    2. In the mass actions toolbar, select Checked.

    3. Continue with the next step.

  5. Select Edit in the mass action toolbar, then click Submit.

  6. In the resulting dialog box, make the following selections.

    • Coding form: select the form that includes the needed tags.

    • Tag group: select the applicable tag group (must be selected by user).

    • Tags: select or clear individual tags. To leave tags that have already been applied unchanged, leave the option box as-is, .

    IMPORTANT!

    • Coding form rules are not enforced when tags are applied in this manner.

    • If you mistakenly select/clear a tag that you do not want to change (the option is no longer ), click Cancel in the Apply Mass Action dialog to preserve all existing tags, then repeat this procedure to apply or clear tags as intended.

  7. Click Apply.

  8. Repeat this procedure for other groups of documents that need to be tagged in the same way.
