Modify Coding Forms

To change the order of coding forms, or to revise or delete an existing coding form:

  1. Get started:

    1. Start Eclipse and log in as an Eclipse administrator.

    2. Open the case for which coding forms are being revised.

    3. In the Case View pane, click Case Administration, and then Coding Forms.

  2. To re-order coding forms (as they are listed in both Case Administration and in the list in Eclipse Record View), drag each form to be moved to its desired location in the coding form list.

  3. To modify an individual form, click it in the list and complete the following steps as needed. (For tips on working in lists, see Use Lists.)

  4. To delete the selected coding form, click , then click Yes in response to the confirmation message.

  5. To revise the selected coding form, click  and make changes as follows:

  6. Fields: Add, remove, or reorder the fields in the form.

  7. Tags: Add, remove, or reorder tags in the form.

  8. Rules: Edit or delete a rule by pointing to the grey bar to the left of the rule and clicking the needed button:

  9. Refer to Create New Coding Forms for details on coding form options.

  10. When finished, click Save.

  11. Notify users of the changes that have been made.
