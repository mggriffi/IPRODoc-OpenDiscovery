Manage Review Passes

Get Started

Complete the following steps to work with review passes in Eclipse:

  1. Log in to Eclipse as an administrator and open the case of interest.

  2. In the Case View pane, click Case Administration, and then Review Pass Management.

  3. In the list of review passes, click the review pass of interest. If the list is long, take any of the following actions to find the needed review pass:

    • Scroll vertically and/or horizontally.

    • Drag the list’s borders to make it wider.

    • Sort the list by clicking the sort button, , and choosing from the different options. Click a selected option again to reverse the sort order.

    • In the search box above the list, start typing any of the following items: the review pass name, the name of the administrator who created/modified the review pass, or part of the created/modified date. This is a plain text search that searches these items in order.

Change Review Pass Definition

After a review pass is defined, you can change any part of its definition.

Note: If batches have already been created, some changes will have no effect on existing batches (for example changing batch size). Changing the search upon which the review pass is based will add documents that were not results of the first search to the review pass.

To change a review pass definition:

  1. Complete Get Started above.

  2. For the selected review pass, click  and make needed changes. For details on options, see Review Pass Basics.

  3. When finished, click Save.

  4. Inform users of changes.

Export Review Pass Details to a .CSV File

To export details (name, creation/modification dates, and so forth) for all review passes defined for a selected case:

  1. Complete Get Started above. (Note that results will be the same whether or not a specific review pass is selected.)

  2. Below the list of review passes, click Export to CSV.

  3. In the Save As dialog box, navigate to an appropriate location for the file, enter a file name, and click Save.

  4. Open the file in an appropriate program.

Delete a Review Pass

To delete a review pass (all batches will also be deleted):

  1. Complete Get Started above.

  2. For the selected review pass, click , then click Yes in response to the confirmation message.

Manage Batches

The Batching tab in Eclipse allows you to perform several batch management activities.

Get Started

For the procedures in this section:

  1. Log in to Eclipse as an Eclipse administrator and open the needed case or batch.

  2. For an overview of batch status, review the Case Details Dashboard as explained in the Eclipse Web End User Help.

  3. Continue with the procedures in the remainder of this section.

Filter or Sort the Batch List

It may be useful to sort or filter the list of batches so that you can evaluate and/or select from a specific set of batches.

Filter the List

To filter Batching tab details:

  1. Log in to Eclipse and open the needed case.

  2. In the Case View pane, click Batches. The Batching tab opens.

  3. Locate the column by which you want to filter the list and click the funnel button , then complete step 4 or 5 as needed.

  4. To select from existing column entries, click one or more options in the top part of the Filter dialog box.

  5. To define more complex criteria, select filtering criteria in the bottom of the Filter dialog box:

    1. Select a needed criterion from the first list. Several options exist, such as Is Equal To, Starts With, or Does not contain.

    2. In the field under the first criterion list, enter the details by which you want to filter.

    3. If the entry in step b includes text for which capitalization is important, enter the details using the needed case and click the  button.

    4. If you want to include a second criterion, select the needed connecting operator, And or Or, and repeat steps a – c in the bottom part of the dialog box. This figure shows filtering by a particular range of batch size:

    5. When the filtering criteria is defined, click Filter.

    6. To filter by an additional column(s), repeat this procedure for the additional column(s).

  6. Click Filter.

  7. To clear filtering:

    1. Locate the filtered column and click .

    2. In the Filter dialog box, click Clear Filter.

    3. Repeat to clear other columns.

Sort the List

To sort details on the Batching tab:

  1. Log in to Eclipse and open the needed case.

  2. In the Case View pane, click Batches. The Batching tab opens.

  3. Locate and click the column heading by which you want to sort. The column will first sort in ascending order. Sorting is indicated by a small triangle above the column name.

  4. To change to descending order, click the heading again.

Mark Batches as Complete

After all documents in a batch have been reviewed (all coding form rules have been met and/or required tags applied), the batch can be marked as complete:

  1. Complete Get Started above to access the Batching tab.

  2. Locate a batch in which all documents have been reviewed. If it is not active, Click the Active option for the batch.

  3. Click Check In.

  4. In the resulting dialog box, click Completed and click OK.

  5. Repeat this procedure for other completed batches.

Split Large Batches

If a batch proves to be too large for a single reviewer, it can be split into smaller batches, with the following conditions:

  • Be aware that if you split a batch created with the Family Field option, family documents will no longer be together in a single batch.

  • If a batch is partially reviewed, only unreviewed documents will be split. The original batch will contain the reviewed documents.

  • If all documents in the batch are unreviewed, all documents will be split and the original batch will be left with zero documents.

  • Split batches are not assigned to a reviewer, even if the original batch was assigned to someone.

  • Split batches are assigned an available number in the numbering sequence.

To split a large batch:

  1. If reviewers might be working in the batches you will be splitting, you may want to inform them that the batches will be changing and that should stop reviewing until changes are made.

  2. Complete Get Started above to access the Batching tab.

  3. Select one or more batches that are to be split in the same way by clicking the leftmost option(s) on the Batching tab as shown in the following figure.

  4. In the batch details (right) pane, click Split Batches.

  5. Delete the default batch size and enter the needed batch size. Minimum batch size is 10.

    6. Note that the batch size displayed is the default batch size, not necessarily the size of the selected batch(es). In the following example, you might want to split the selected batch(es) into ten 100-document batches.

  6. Click Create batches and wait as the batches are created.

  7. Assign the new batches to reviewers and/or take other batch management actions.

  8. Inform reviewers about the batches and provided needed instructions.

Save Batch Details to a .CSV File

It may be useful to save batch details (contents of the Batching tab) to a spreadsheet. To save Batch Management window details to a .CSV file:

  1. Complete Get Started above to access the Batching tab.

  2. As needed, filter the tab details by batch set and/or status. Click at the bottom of the tab to ensure all data are refreshed.

  3. Click Export to CSV, then Run.

  4. In the Save dialog box, navigate to an appropriate location for the file, enter a file name, and click Save.

  5. Open the file in a spreadsheet or other appropriate application.

Delete Specific Batches

To delete batches:

  1. Complete Get Started above to access the Batching tab.

  2. Locate one or more batches to be deleted; select them by clicking the leftmost option(s) on the Batching tab.

  3. When all batches have been selected, click Delete this batch (or selected batches) in the right pane.

  4. In response to the confirmation message, click Yes. All documents in the deleted batch are indicated in the review pass as unbatched.

  5. Inform users of changes regarding available batches.
