Evaluate Case Records

As part of the following case activities, you have the ability to view case record details and make certain record changes in Eclipse Administration:

  • After importing case data

  • After migrating a case

  • As part of case management

Regardless of the specific activity, the record evaluation and tasks are the same, as explained in this section.

Get Started

To evaluate records as part of your Eclipse Administration activities, complete the following procedure:

  1. Start Eclipse Administration and log in as an Eclipse administrator.

  2. Complete one of the following procedures:

    • Import case data and/or files as explained in Import Case Data or Import Text Files.

    • Migrate a case as explained in Migrate Cases from a Third-Party Database.

    • Manage the case as explained Manage a Case.

  3. If you are importing case data, complete the following after you have imported the case:

    1. In the Import Data window, click the Step 4. Verify tab.

    2. Go to step 6 below.

  4. If you are migrating a case, complete the following after you have migrated the case:

    1. In the Migrate from Concordance window, click the Step 4. Verify tab.

    2. Go to step 6 below.

  5. If you are managing an existing case:

    1. In the Case Management screen, select the needed managing client and case.

    2. Click the Database Records tab.

    3. Go to the next step.

  6. Click List All Records. Or, list selected records based on a search by clicking Load Search.
