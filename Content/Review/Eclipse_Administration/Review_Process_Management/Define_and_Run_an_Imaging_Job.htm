Define and Run an Imaging Job

To create and add images to your Eclipse case:

  1. Get started:

    1. Review the Understanding Imaging to ensure all requirements have been met.

    2. Start Eclipse and log in as an Eclipse administrator.

    3. Open the case for which image files are being created.

  2. In the Case View pane, click Case Administration, then Imaging.

  3. Define the imaging job as described in the following table.

    4. Image Job Options

    Option

    Description

    Output Options

    Imaging Set

    Select the saved search containing the documents to be processed.

    Select Mode

    Select one of the following options for importing the image files:

    Overlay

    Select to add new images (for existing and missing images)

    Overlay Replace

    Select to delete existing images before adding new images (if images exist), or add new images (if images are missing).

    Color Options

    Select the method by which colors will be processed:

     

    Black and white (1 bit)

    Renders as Group 4 TIFF1

     

    Grey scale (8 bit)

    Renders as LZW TIFF

     

    Color (8 bit)

    Renders as LZW TIFF

     

    True color (24 bit)

    Renders a JPG

     

    Use eCap project default

    Use the color settings of the selected eCapture project.

    Connection

    eCapture Project

    Select the eCapture project to be used for the Eclipse imaging job. Files will be processed using all eCapture project settings (except color settings if you selected color options for this job). For details on eCapture project options, see the eCapture User Guide.

    Custom Placeholder Fields

    Original File Name

    Select the fields containing the Original File Name and Location, to be included in placeholder files for documents with no associated native files.

    Original File Location

    1Color rendering is explained on many public websites. Refer to those resources if you want details on color settings.
