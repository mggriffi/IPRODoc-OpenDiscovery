Evaluate Imaging Errors

After an imaging job completes, complete the following steps if errors are indicated:

  1. Run the search on which the imaging job was based.

  2. In the Search Results tab, add the imaging fields (listed in the following table) to the case table (grid), along with other needed fields.

    Field

    Description

    ENT IMAGING ERRORS

    Errors that occur during imaging

    ENT IMAGING FLAGS

    QC flags from eCapture

    ENT IMAGING JOB

    Name of the imaging job

    ENT IMAGING STATUS

    Status of the imaging job

  3. Organize, sort, and/or filter the table as needed. For details on working in the case table, see Eclipse Web End User Help.
