Before you begin

Planning will bring you the greatest success with custom search indexes for each case. To familiarize yourself with needed information and available options, “walk through” the index management function as follows:

  1. Ensure that the Eclipse Analytics Server is installed and accessible by the Eclipse Data Server.

  2. You need to understand your case data and reviewer tasks. Review cases and discuss with reviewers the types of searches that will assist them as they do their jobs.

  3. Start Eclipse Administration and log in as an Eclipse administrator.

  4. On the Case Management ribbon, select Eclipse Analytics > Manage Analytics Indexes.

  5. Select the needed managing client and case.

  6. Take the following actions and make a note of the information you will need.

    1. Click Create New Index: Make a note of the basic index definition. See Create an Index for details. When finished, click Cancel.

    2. Click Manage Index Profiles. A profile is the first step to creating a custom index. Although the dialog box is dimmed, you can review the fields and entries. See Create an Index Profile for details. When finished, click Close.

    3. Review the Index Status (bottom) portion of the Manage Indexes window. See Build an Index for details on these actions.

  7. With this preview in mind, gather needed information and perform preliminary actions (such as creating searches in Eclipse to be used for your training set and searchable set of documents).

  8. Continue with the procedures explained in this section.
