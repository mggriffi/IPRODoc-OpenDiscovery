Build an Index

To make an index available in Eclipse:

  1. Complete Create an Index.

  2. In the Managing Indexes window, click the index to be built in the index list.

  3. Click Build. This button will be active for indexes that have not yet been built or if an index definition or case data have changed.

  4. Wait as the index is processed. The amount of time required for this and the following steps will depend on the size of the training and searchable data sets, plus various profile, system, and other factors. When the build is complete, a success message displays.

  5. To enable conceptual searching for the index, click Enable Queries. Optionally, select the Enable Queries on Startup option.

  6. If any problems occur, click View Log File and evaluate the build details.

  7. Alert your users to the availability of the new conceptual search index. Explain how they can use it as they review documents.
