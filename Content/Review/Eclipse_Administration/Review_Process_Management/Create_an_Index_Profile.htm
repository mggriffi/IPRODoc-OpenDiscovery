Create an Index Profile

The index profile allows you to specify various options for building the index, such as number of dimensions, pruning values, processes, and so on. To create an index profile:

  1. Get started:

    1. Start Eclipse Administration and log in as an Eclipse administrator.

    2. On the Case Management ribbon, select Eclipse Analytics > Manage Analytics Indexes.

    3. In the Manage Indexes window, select the needed managing client and case.

  2. Click Manage Index Profiles.

  3. Click Create New Profile.

  4. Enter the following basic information in the top portion of the dialog box.

    5. Click here for Index Profile details

    Item

    Description

    Profile Name

    Required. Enter a name for the index profile.

    Content Analyst Server

    Required. Select the Content Analyst Server to be used. If the server is not listed:

    1. Click  next to the server list.

    2. Click Add Server.

    3. Enter the URL for the needed Content Analyst Server and click OK.

    4. Click OK.

    5. Select the server in the list.

    Dimensions

    Enter the dimension to be used in the conceptual search, typically a number from 100 to 300. The default and recommended dimension setting is 100.

    “Dimensions” are an aspect of the conceptual search calculation, representing the concept space into which documents are mapped when an index is built.

    When considering this entry, note:

    • A higher dimension will yield a greater the number of search results, because conceptual values are increased and relationships are refined. It also increases query processing time and utilizes more of the query server’s resources, particularly RAM.

    • A lower dimension setting may lead to less accurate search results.

    • An entry of 0 (zero) instructs the index to select a reasonable value.

  5. Create New Filter(s): To create filters for repeated content:

    1. In the Index Profiles dialog box, click Create New Filter.

    2. Name: Enter a unique name for the repetitive text you are defining.

    3. Type: Select Literal or Regular Expression to specify the type of expression you are defining. Use regular expressions if you are familiar with the syntax and construction of such expressions.

    4. Pattern: Enter the content to be filtered (literal or regular expression). Check your entry carefully to ensure it defines the pattern you want filtered.

    5. When finished, click OK.

    6. Repeat steps a-e to define other content to be filtered.

    7. When finished, click OK.

  6. When your index profile is complete, click Save Modifications, then Close Window.
