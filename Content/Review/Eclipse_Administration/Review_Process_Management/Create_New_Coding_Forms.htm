Create New Coding Forms

To create coding forms for a case:

  1. Get started:

    1. Review Understand Coding Forms.

    2. Start Eclipse and log in as an Eclipse administrator.

    3. Open the case for which coding forms are being created.

    4. In the Case View pane, click Case Administration, and then Coding Forms.

  2. In the Case Administration tab, click F.

  3. Enter a name for the form and complete the following steps as needed.

  4. Optional: If no rules will be defined for the form and you want reviewers to be able to automatically save field edits and/or tagging, select Enable Auto Save.

  5. Permissions: If the form should be restricted to certain user groups:

    1. Click .

    2. Select needed groups and click OK.

      TIP: To make a coding form available to all groups that are currently defined for the case, click Select All. The form will be unavailable for groups added after this selection (unless you add them).

  6. Fields: Select the fields to be displayed on the Record View pane:

    1. Expand the Fields section of the Coding Form configuration area.

    2. To select all fields, click .

    3. To select individual fields, double-click the fields in the Available Fields list.

    4. To select multiple items, click a field name, then Shift+click or Ctrl+click additional fields to choose a contiguous or non-contiguous set, respectively. When needed fields have been selected, click .

  7. Field order: When needed fields have been selected, organize them in a way to best meet your users’ needs:

    1. In the Selected Fields area, click a field to be reordered.

    2. Click Up or Down until the field is in the desired location.

    3. Repeat these steps to reorder other fields.

  8. Tags:

    1. Expand the Tags section of the Coding Form configuration area, then click Add/Remove.

    2. Select the tag groups and tags to be included. Note: If a tag group has the Exclusive rule, all tags in the group must be selected.

    3. Click OK in the Select Tags dialog box.

  9. Tag group order: When needed tags have been added, organize them in a way to best meet your users’ needs:

    1. In the Tags area, click a tag group to be reordered.
    2. Click Up or Down until the tag group is in the desired location.
    3. Repeat these steps to reorder other tag groups.

  10. Rules: See Define Coding Form Rules for details on setting rules as part of your coding form definition.

  11. When finished, click Save.

  12. Repeat this procedure to configure other coding forms.

  13. Explain coding form use to users and/or add applicable case instructions.
