Manage the TAR Review Process

As the workflow diagram on Manage TAR Projects indicates, various tasks are required after the TAR project is created. These tasks, covered in this section, focus on Eclipse components created as part of the TAR project, such as coding forms and review passes.

Important: All TAR-related components are defined as needed for proper TAR processing; it is recommended that the configurations of these components not be changed except as described in the following procedures.
