Change TAR Configuration

To change a TAR project:

  1. Get started:

    1. Start Eclipse and log in as an Eclipse administrator.

    2. Open the case for which a TAR project is being revised.

    3. In the Case View pane, click Case Administration, and then TAR Project Setup.

  2. Click and make changes as follows:

    • Change Training Set batch size.

    • Change Validation Rounds sampling type and/or associated settings.

    • Change Certification Set sampling type and/or associated settings.

    3. For details on these aspects of a TAR project, see applicable steps in TAR Project Configuration.

  3. Click Save.

These changes will apply only to unstarted activities. For example, changes to the Training Set batch size will apply only if the training round has not been started. Changes to Validation Rounds sampling will apply to rounds that start after the changes are made.

Delete a TAR Project

A TAR project can be deleted from the TAR project (summary) list or deleted completely. Deleting a TAR project completely deletes all aspects of the project, including associated tags (whether applied or not), coding form, fields, category set, and review passes will be deleted.

To delete a TAR project:

  1. Get started:

    1. Start Eclipse and log in as an Eclipse administrator.

    2. Open the case containing the TAR project to be deleted.

    3. In the Case View pane, click Case Administration, and then TAR Project Setup.

  2. Click the project to be deleted.

  4. Click Delete.
