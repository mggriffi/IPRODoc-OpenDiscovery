Eclipse Administration FAQs

Can Eclipse create OCR?

OCR can be generated for multiple documents at a time through Eclipse Admin. Additionally, during the Streaming process, you can choose to create OCR for documents that do not contain extractable text so that this is not a manual process you are having to perform once documents are loaded into Eclipse.

What transcript formats will Eclipse handle?

The supported formats are ASCII, PTF, PCF, TRN, and XML.

What’s the throughput rate (GB / Hr)?

On average, the throughput rate is 5GB / Hour which includes the copy process, metadata extraction, OCR, export and import into Eclipse.

What internet browser is needed for Eclipse?

Currently Eclipse only runs on Internet Explorer.

Do we have to tell the analytics engine what words we are interested in?

The analytics process does not require keywords for Clustering, but allows for any combination of keywords, phrases, or even entire documents for Categorization.

Can we secure a subset of documents and work product from experts or the client?

In order to achieve this, you can create a separate database with a subset of your data. Eclipse allows for easy movement of both records and work product between databases.

Who makes your document viewer?

The viewer is Oracle’s Outside In (Stellant). As we migrate to HTML5, this will be changing in future releases.

In Visual Searching, how do I make a field eligible for use on a chart axis?

The “Filter Field” attribute must be assigned to a field before it is eligible for use on a chart axis. 

What is an Code name?

A code name, serves as a primary organizer for one or more related cases, as a shortened version of a very long case name, or for any purpose that will help you organize and recognize cases across all products.

What is the difference between a Client and a Management Client?

Managing clients provide a basis for case organization and the relationship between cases and reviewers. For service providers, managing clients are the companies and firms with which the provider does business. For law firms, the firm itself is the managing client.  A secondary or subordinate client provides further level of identification. For law firms this may be the same as the managing client or may be the individual clients; for service providers, different secondary clients may also be appropriate. 

Can I export to Disk instead of going directly to Eclipse?

Yes, when creating or modifying an export series or set, you may choose the option to Export to Disk. 

How do I apply a time zone offset in Eclipse?

Time zone offsets can be set in Eclipse Administration by using the Modify Case Option within Case Management. 

Is there a way to automatically mark a batched document as ‘Reviewed’ when applying specific tags?

Yes, this can be accomplished by using coding form rules. 

Can I import an existing production into Eclipse?

Yes, you can do this by selecting the Import Production History option in the Eclipse Admin desktop application.

How do I import text files when the path is in an ASCII load file?

During the import phase in "Step 2: Load a Case Definition" where users are mapping fields from the load file to Eclipse Case fields, there is an option to modify the text path. Use this option to change the path to the correct location where data is currently stored on the network.

What is the recommended number of Search Indexes to use for my Eclipse case?

This largely depends on how many documents are expected in a case. Generally, the default of 10 indexes is fine, however, if the case will grow to a very large size, increasing this number can help with search performance.

If you delete a Permission Template in an Eclipse case, does it remove the template from the Administration page?

Yes, changes made to the permission template at a case-level are also removed from the system-level Administration page.

What is the best way to update the native file paths in my case if I have moved them?

The best/safest way to update native file paths would be to create an overlay load file, and then import this via Eclipse Admin desktop. Another option would be to use the search and replace function of mass action in Eclipse Web to modify the path.
