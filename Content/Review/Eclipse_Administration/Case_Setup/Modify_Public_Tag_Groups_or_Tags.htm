Modify or Delete Tag Groups

To revise or delete an existing tag group:

  1. Get started:

    1. Start Eclipse and log in as an Eclipse administrator.

    2. Open the needed case.

    3. In the Case View pane, click Case Administration and then Tag Palette.

  2. In the list of tag groups, click the tag group to be revised or deleted. For tips on working in lists, see Use Lists.

  3. To delete the selected tag group, click , then click Yes in response to the confirmation message.

    Tip: If a group’s tags have been applied, the group cannot be deleted (the  button will not be available).

  4. Click  and continue with the following steps as needed.

  5. To rename the tag group:

    1. Click the tag group name under the Modify Tag Group heading.

    2. Click Rename in the Tag Group area.

    3. Enter the new group name (up to 100 characters) and click Update.

  6. If needed, make changes to tag group rules.

  7. If tags need to be changed, continue with the following procedure. If not, click Save.

Modify or Delete Tags

To revise or delete public tags:

  1. Complete the introductory steps in Modify or Delete Tag Groups to access tag configuration.

  2. With the needed tag group selected, click  and complete the following steps.

  3. To add new tags:

    1. Click Create Tags.

    2. Add the tags (one per line, up to 250 characters for each tag name) and click Add when finished.

    3. Continue with other changes or skip to step 6 if changes are complete.

      Note: Nested (child) tags can also be added to tags, with certain restrictions:

      • If a tag has no child tags and has not yet been applied to documents, then child tags can be added.

      • If a tag has no child tags but has been applied to documents, then child tags cannot be added.

      • If a tag has child tags, additional child tags can be added, whether or not any of the tags have been applied.

  4. To rename a tag:

    1. Click the tag name in the Modify Tag Group area, then click Rename.

    2. Revise the name (up to 250 characters per name) and click Update.

    3.  Continue with other changes or skip to step 6 if changes are complete.

  5. To delete a tag:

    1. Click the tag name in the Modify Tag Group area, then click Delete Tag.

    2. In response to the confirmation message, click Yes.

    3. Continue with other changes or skip to step 6 if changes are complete.

      Note: Tags that have been applied (listed in red) cannot be deleted.

  6. When all changes are complete, click Save.

  7. Notify users of tag changes and provide instructions as needed
