Globally Replace Image Paths

The Case Management window in Eclipse Administration allows Eclipse administrators to replace image paths (original images or production images). To replace image paths for your (entire) case:

  1. Get started:

    1. Verify the details needed for this operation (path to be found and needed replacement path).

    2. Start Eclipse Administration and log in as an Eclipse administrator.

    3. On the Case Management ribbon, select Case Management > Case Management.

    4. In the Case Management window, select the needed managing client and case.

  2. Click the Global Replace tab.

  3. In the list of Available Fields, select one of the following options:

    • <Image Path> to replace the path for original case images

    • <Production Image Path> to replace the path for produced images

  4. In the Search For field, enter the path to be replaced.

  5. In the Replace With field, enter the replacement path.

  6. Click Go. A message will display when the search/replace action is complete.

When finished, continue with other administrative activities.

Note: If you need to globally replace field content, see Eclipse Web End User Help. Editing field content for multiple records is a “mass action” available in Eclipse Web.
