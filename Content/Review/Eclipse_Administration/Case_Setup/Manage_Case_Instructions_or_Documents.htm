Add Case Instructions or Documents

To add case instructions or non-discovery documents to a case:

  1. Prepare or gather needed documents. Note:

    • Ensure documents are in a network location accessible by Eclipse users.

    • Keep in mind that to view case instructions or alerts from Eclipse, users must have the corresponding program.

    • If several documents will be present for users, give some consideration to naming consistency.

  2. Start Eclipse Administration and log in as an Eclipse administrator.

    Note: only members of the Super Admin group can add case documents.

  3. On the Case Management ribbon, select Case Management > Case Management.

  4. In the Case Management window, select the needed managing client and case.

  5. Click the Case Details tab.

  6. To add case instructions:

    1. In the Case Instructions area, click .

    2. Navigate to and select one or more instructions files.

    3. Click Open.

  7. To add case documents:

    1. In the Case Documents area, click .

    2. Navigate to and select one or more case document files.

    3. Click Open.

  8. Inform users that documents may be present and they should check for new documents in the Eclipse Case Details Dashboard whenever they log in to Eclipse.

Edit Case Instructions or Documents

Modify a case instruction or document by making changes to it using the native application. To ensure users know that a new document exists, take one of the following actions:

  • Save the modified file with a new name and add it to the case using the previous procedure.

  • Save the file with its existing name and add an alert that tells users the file has been modified.

Delete an Instruction or Case Document

To delete a case instruction or document:

  1. Get started:

    1. Start Eclipse Administration and log in as an Eclipse administrator.

    2. On the Case Management ribbon, select Case Management > Case Management.

    3. In the Case Management screen, select the needed managing client and case.

  2. To delete an instruction or document:

    1. Click the Case Details tab.

    2. In the Case Instructions or Documents tab, click  next to the document to be deleted.

    3. In response to the confirmation message, click OK.

  3. Notify users of the changes that have been made.
