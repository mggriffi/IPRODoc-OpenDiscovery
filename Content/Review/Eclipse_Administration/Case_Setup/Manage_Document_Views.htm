Manage Document Views

In Eclipse, a menu (shown in the figure below) allows users to choose from the following views:

  • All documents in the open case
  • When a batch has been opened: the user’s unreviewed documents, reviewed documents, or the user’s batch

Administrators have the option to limit one or more of these views to certain review groups and/or organize their appearance in Eclipse. For example, you may not want users to view all case documents. In this case, you would limit the All Documents view to a group that includes only administrators.

To re-order or secure document views:

  1. Get started:

    1. Start Eclipse Administration and log in as an Eclipse administrator.

    2. On the Case Management ribbon, select Case Management > Case Management.

    3. In the Case Management screen, select the needed managing client/case.

  2. Click the Document Views tab. (If it is not visible, click  on the right side of the screen’s tabs.)

  3. To change the order of views in Eclipse:

    1. Click a view name.

    2. Click the Up or Down button until the view is in the needed location.

    3. As needed, repeat for other views.

  4. To limit a view to specific review groups:

    1. Click  corresponding to the view that will be restricted.

    2. In the Select Groups dialog box, select the group(s) that should have access to the view.

    3. When finished, click OK. Only members of the selected groups will be able to use the view.

    4. As needed, repeat for other views.

  5. To change restrictions, click the lock button and revise your choices in the Select Groups dialog box.
