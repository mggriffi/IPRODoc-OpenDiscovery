About Review Cases

A case can be created in for Eclipse® by (Review case) or for Streaming discovery (Processing & Review case).

With either method, the case is based on a default or user-selected Eclipse template (.CSE file), which may include any or all of the following:

  • Fields

  • Tags

  • Redactions

  • Noise words

  • Index settings

  • Coding forms

  • Relationships

  • Case options (tagging and date format)

The default template used by is DefaultReviewCaseSettings.cse, which is located by default in the system’s Program Files\Ipro Tech\Web\ADDServices\App_Data folder.

Check/Revise the Case

Based the template selected in , the case definition may be complete or may need to be completed or revised in Eclipse. Therefore, if you will be working with a case created in , complete the following steps:

  1. Start Eclipse Administration and log in as a user who is a member of the Super Admin group.

  2. On the Case Management ribbon, select Case Management > Case Management.

  3. Select the case.

  4. Review the following details and make changes as needed:

    • Case details

    • Database fields and field definitions

  5. Log into Eclipse (Web) as a member of the Super Admin group and open the case.

  6. Review the case definition by checking all Case Administration settings, starting with those defined for the case (see list above). Make changes as needed.
