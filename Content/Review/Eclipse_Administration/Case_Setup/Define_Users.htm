Create New Users Manually

To create Eclipse users manually:

  1. Get started

    1. Ensure that you have needed details for the users you will be creating. See the table in step 3.

    2. On your site’s Eclipse website, log in as a user who is a member of the Super Admin group.

    3. Click the Eclipse button, , and select Administration.

    4. If the System Administration pane is collapsed, click the pane title to expand it.

    Trusted IPs

    To limit a user’s access to Eclipse to one or more specific computers, enter each computer’s IP address (one address per line) in this field. Note:

    • If no IP addresses are entered, users will be able to log in to Eclipse from any computer (to the extent allowed by any security for the network in which Eclipse resides, for example, company firewall).

    • Use standard IP address format, ###.###.###.###, with the address’s actual number of digits per block. (For example, 123.456.78.9.)

    • Eclipse validates IPv4 and IPv6 protocols.

View, Edit, or Delete Users

Note: Before editing or deleting an Eclipse user, remember that manages all users, and an “Eclipse” user may also be working in (or Allegro). Verify particular user’s activities before you edit or delete them as a user.

 

To view, modify, or delete users:

  1. Get started:

    1. On your site’s Eclipse website, log in as a user who is a member of the Super Admin group.

    2. Click the Eclipse button, , and select Administration.

    3. If the System Administration pane is collapsed, click the pane title to expand it.

  2. Click Users.

  3. Review the list in the Users tab. To find a specific user in a long list, see the tips in Use Lists.

  4. To view details of a particular user, click the name in the list; details appear on the right.

  5. To edit user details:

    1. Click the user in the list.

    2. Click  and make needed changes.

    3. Click Save.

  6. To delete a user:

    1. Click the user type in the list.
