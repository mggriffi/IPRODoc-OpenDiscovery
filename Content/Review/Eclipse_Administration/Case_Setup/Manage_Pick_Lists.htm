About Pick Lists

A pick list is a set of items (terms) from which a reviewer can choose when editing a field in Record View. The following figure shows a typical pick list for the CustodianName field.

Because the set of items in a pick list may be appropriate for more than one field in a case, the definition of pick lists is separate from the definition of the case’s fields. Pick lists are defined as explained in this section and selected as part of field definition or management (see Step 3: Define Database Fields).

If a field is modified to be a pick list after data is imported, all data for the field will remain intact. Reviewers needing to revise the field will select an item from the pick list.

Plan Pick Lists

As with all Eclipse tasks, proper planning will help ensure that the pick lists you create are useful to your reviewers. For each pick list, consider such factors as:

  • Can the pick list be designed to apply across multiple fields (such as a field for which Yes and No are needed entries) or is it specific to only one or a few fields?

  • Have all needed choices been identified?

  • What kind of naming/terminology will be most useful to reviewers? (For example, for names, should first and last names be included or just last names?)

  • Pick lists items are listed in alphabetical order. If one item will be selected the majority of the time, consider preceding its name with a special character (such as an asterisk or underscore) to keep it at the top of the list.

Create Pick Lists

To create a pick list:

  1. Get started:

    1. Review About Pick Lists above.

    2. Start Eclipse and log in as an Eclipse administrator.

    3. Open the case for which pick lists are being created.

    4. In the Case View pane, click Case Administration, and then Pick List as shown in this figure.

  2. In the Case Administration tab, click .

  3. In the Pick List area, enter a name for the list.

  4. In the Terms field, enter items to comprise the list, one item per line. Maximum size for each term is 1,000 characters.

  5. When finished, click Save.

  6. Repeat this procedure to create other pick lists.

  7. Create or modify database fields that should be pick lists. (See references in About Pick Lists.)

  8. Explain pick list details to reviewers who will be editing fields in Eclipse.

Modify Pick Lists

Complete the following steps to modify or delete a pick list through case administration.

Note: Users with appropriate permissions can also add items to a pick list while they are working in Record View.

  1. Get started:

    1. Start Eclipse and log in as an Eclipse administrator.

    2. Open the case for which the pick list is being modified.

    3. In the Case View pane, click Case Administration, and then Pick Lists.

  2. In the list of pick lists, click the one to be modified. For tips on working in lists, see Use Lists.

  3. Review the fields in the Assigned Fields list (on the right) and verify that the changes you will make apply to all of the fields.

  4. To delete the selected pick list, click , then click Yes in response to the confirmation message.

  5. To revise the pick list, click  and make needed changes. Refer to Create Pick Lists for details on pick list configuration.

  6. When finished, click Save.

  7. Notify users of the changes that have been made.
