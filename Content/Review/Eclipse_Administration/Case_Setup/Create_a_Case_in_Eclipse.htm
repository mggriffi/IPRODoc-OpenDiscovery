Create a Case in Eclipse

This page focuses on creating a new case directly in Eclipse.

If you are new to case management, the following information should help you set up cases in Eclipse Administration. Experienced case administrators may find it very easy to work through the steps (tabs) without referring to this information.

Creating a case in Eclipse includes five distinct steps, as outlined on the Create Case window:

  • Step 1: Set Up a Case

  • Step 2: Load a Case Definition

  • Step 3: Define Database Fields

  • Step 4: Select Case Options

  • Step 5: Finish Case Creation

 

Important: You cannot save a portion of a case definition (such as your entries for step 1). If you exit Eclipse Administration before completing all steps, you will lose your work.
