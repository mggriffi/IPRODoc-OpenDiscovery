Add Case Alerts

To add alerts to a case:

  1. Gather needed information for the alert.

  2. Start Eclipse Administration and log in as an Eclipse administrator.

  3. On the Case Management ribbon, select Case Management > Case Management.

  4. In the Case Management screen, select the needed managing client and case.

  5. Click the Alerts tab. (If it is not visible, click  on the right side of the screen’s tabs.)

  6. Click New Alert.

  7. Define the new alert:

    1. Enter an alert name to identify the nature of the alert. If several alerts will be present for users, give some consideration to naming consistency.

    2. Enter the alert details in as concise a manner as possible. The message is limited to 500 characters.

    3. Select a start date and time. The alert will not appear in Eclipse until that date/time.

    4. Select an ending date and time. The alert will be removed from Eclipse after that date/time.

      Time format will match the format selected for Windows. To change time, click on a time component (such as the hour) and type the needed detail.

    5. Select a priority level, which ranges from Low to Critical.

    6. When all entries are complete, click OK. The alert will be available to all users assigned to the case, as indicated by the open lock, , on the Alerts tab.

  8. To restrict an alert to certain review groups:

    1. Click the alert’s  button.

    2. Select the groups that should have access to the alert, then click OK.

      To make an alert available to all groups that are currently defined for the case, click Select All. The alert will be unavailable for groups added after this selection (unless you add them).

  9. Repeat this procedure to configure other alerts.

  10. Inform users that alerts may be present and they should check for alerts in the Eclipse Case Details Dashboard whenever logging into Eclipse.

Modify an Alert

To modify an alert:

  1. Start Eclipse Administration and log in as an Eclipse administrator.

  2. On the Case Management ribbon, select Case Management > Case Management.

  3. On the Case Management window, select the needed managing client and case.

  4. Click the Alerts tab. (If it is not visible, click  on the right side of the window’s tabs.)

  5. Click  next to the alert to be revised and make changes in the New Alert dialog box. Refer to Add Case Alerts for details on this dialog box.

  6. If needed, restrict the alert to certain review groups:

    1. Click the Security (lock) button (whether it is open or closed) corresponding to the form being revised.

    2. Select or clear groups as needed, then click OK.

  7. Notify users of the changes that have been made.

Delete an Alert

To delete an alert:

  1. Click the Alerts tab. (If it is not visible, click on the right side of the screen’s tabs.)

  2. Click next to the alert to be deleted.

  3. In response to the confirmation message, click OK.

  4. Notify users of the changes that have been made.
