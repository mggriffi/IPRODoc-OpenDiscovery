Redactions in New Cases

Newly created cases include redactions as follows:

  • Cases created from an existing case include the redactions defined for that case.

  • Cases created from a .CSE file, may include the redactions defined in the .CSE file (including the redactions is optional).

  • Cases created based on a blank template include the default redaction.

Changes to redactions can be made after case review begins (except that redactions that have been applied cannot be deleted). For a particular case, redactions may not be needed, in which case you can remove the default redaction.

Plan Redactions

Before you begin, plan your redaction palette:

  • Redaction labels and naming consistency

  • Redaction color and font color (legibility and color coding)

  • Redaction order (most frequently applied redactions are typically first)

    Note: By default, redactions are listed in alphanumeric order in the Case Administration tab and in ID order in the Image tab.

Also consider the information users will need in order to properly use redaction and create appropriate case instructions.

Create New Redactions

To create redactions for a case:

  1. Get started:

    1. Start Eclipse and log in as an Eclipse administrator.

    2. Open the case for which redactions are being created.

    3. In the Case View pane, click Case Administration and then Redaction Categories.

  2. In the Case Administration tab, click .

  3. In the Redaction Category area, click the ID number corresponding to the position in the redaction list (in the Image tab) that the new redaction should be.

  4. Enter the redaction label and select its color and font color. The following figure shows a typical redaction configuration.

  5. Evaluate the sample text to ensure that the label is legible.

  6. When finished, click Save.

  7. Repeat this procedure to create other redaction categories.

Modify Redactions

To revise or delete an existing redaction:

  1. Get started:

    1. Log in as an Eclipse administrator.

    2. Open the case for which redactions are being created.

    3. In the Case View pane, click Case Administration, and then Redaction Categories.

  2. In the list of redactions, locate and select the redaction to be revised or deleted. For tips on working in lists, see Use Lists.

  3. To delete the selected redaction, click , then click Yes in response to the confirmation message.

    Tip: If a redaction has been applied, it cannot be deleted; a message will alert you if you attempt to delete a redaction that has been applied.

  4. To modify the redaction, change the label and/or colors as needed.

  5. When finished, click Save.

  6. Repeat these steps to modify or delete other redactions.
