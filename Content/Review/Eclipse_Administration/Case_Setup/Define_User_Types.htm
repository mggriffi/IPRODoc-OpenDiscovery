Define New User Types

Eclipse allows users to be identified by type, such as “Billable” and “Not Billable.” Create user types if needed to track user activities.

  1. Get started:

    1. For best results, consider a naming convention for user types.

    2. Log in as a user who is a member of the Super Admin group.

    3. Click the Eclipse button, , and select Administration.

    4. If the System Administration pane is collapsed, click the pane title to expand it.

  2. Click User Types, then .

  3. Enter a unique user type name and optionally notes. Notes are searchable in the User Type list, which can be useful.

  4. Click Save.

View, Edit, or Delete User Types

To view, modify, or delete user types:

  1. Get started:

    1. Log in as a user who is a member of the Super Admin group.

    2. Click the Eclipse button, , and select Administration.

    3. If the System Administration pane is collapsed, click the pane title to expand it.

  2. Click User Types.

  3. Review the list in the User Types tab. To find a specific user type in a long list, see the tips in Use Lists.

  4. To view details of a particular user type, click the name in the list; details appear on the right.

  5. To edit user type details:

    1. Click the user type in the list.
