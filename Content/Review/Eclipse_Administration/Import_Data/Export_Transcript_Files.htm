Export Transcript Files

Transcripts can be easily exported from Eclipse to .PTF files; any annotations that have been applied will be included.

 

Related pages:

  • Import Transcript Files

  • Manage Transcripts

 

To export transcripts:

  1. Complete Get Started.

  2. In the Transcript list, right-click the transcript to be exported and select Export Transcript.

  3. Navigate to an appropriate folder, enter a name for the transcript file, and click Save. The transcript is exported to a .PTF file.

  4. Repeat these steps to other transcripts.
