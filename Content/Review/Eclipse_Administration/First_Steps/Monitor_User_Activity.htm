Monitor User Activity

Eclipse allows administrators to monitor user connections and also to generate the User Billing report, which lists active users and the amount of time they have been active for a selected time period.

Monitor Users

About User Connections

The Eclipse User Connections feature allows administrators to monitor current Eclipse users, to communicate with users spontaneously/directly, and to disconnect users if needed. For example, an unexpected maintenance issue may require that everyone log out of Eclipse. You can send a message directly to users and then disconnect anyone who does not log out.

Getting Started

To access user connection details:

  1. On your site’s Eclipse website, log in as a user who is a member of the Super Admin group.

  2. Click the Eclipse button, , and select Administration.

  3. If the System Administration pane is collapsed, click the pane title to expand it.

  4. Click User Connections.

Send User Messages

To send a message to users:

  1. Complete Getting Started above.

  2. Select the users who should receive the message by clicking the option boxes next to user names.

  3. Enter a short message in the message box and then click Send to Selected. Users will receive a pop-up message similar to the following example:

  4. Monitor message status on the right side of the user connections table. The status will be “Waiting” until users click OK in the message box.

  5. Click the refresh button, , to ensure status is up to date.

Disconnect Users

To disconnect users:

  1. Complete Getting Started above.

  2. If possible, send users a message (see previous procedure) to give them time to save their work and log out.

  3. Select the users to be disconnected by clicking the option boxes next to user names.

  4. Click Disconnect Selected. Users are returned to the Eclipse log-in window.

Generate a User Billing Report

The User Billing Report is an administrative report that provides the following information for a selected time period:

  • User name

  • Managing client

  • Date user was created or enabled in Eclipse

  • Number of active days in the selected time period

    Note: Users are counted as “Active” for every day or part of a day (any amount of time) they are enabled in Eclipse during the selected time period. That is, if the Enabled option is selected for a user for any amount of time in a day, that user is reported as active for one day.

To generate a User Billing Report:

  1. On your site’s Eclipse website, log in as a user who is a member of the Super Admin group.

  2. Click the Eclipse button, , and select Administration.

  3. If the System Administration pane is collapsed, click the pane title to expand it.

  4. Click User Billing Report.

  5. Enter Start and End dates to define the scope of the report, or click  to select the dates on a calendar. When finished, click OK.

  6. Evaluate report details.

  7. Optional: Save the report to a .CSV or .XLS file:

    1. At the bottom of the report area, select Export to Excel or Export to CSV.

    2. Click Run.

    3. Complete the Save As dialog box and click Save.

    4. Open either type of file in a spreadsheet or other appropriate program.
