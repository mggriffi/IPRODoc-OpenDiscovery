Get Started with Eclipse Administration

Eclipse Administration is intended for the employees who will administer Eclipse within an organization. As an Eclipse administrator, you are responsible for all aspects of user and case management, reviews, and production.

Eclipse allows more than one individual to be assigned administrative duties, so tasks can be shared.

Administrators are expected to have basic knowledge and understanding of:

  • The legal industry and discovery process.

  • Your organization and how Eclipse will be used within it.

  • Windows administrative functions: areas such as networks, client-server issues, and Windows Active Directory, as they pertain to your organization.

  • Since Eclipse works in concert with Microsoft SQL, the requisite SQL knowledge and skills are required.

Administrative Benefits

Case creation and management

Eclipse Administration is used for the following tasks:

  • Basic definition of each case and its database fields

  • Import of case files

  • Addition of other case-related documents, including transcripts, instructions, alerts, and non-discovery documents

  • Definition of case-specific reports

Dashboard

The Case Details Dashboard allows easy access for administrators and others to visually track the progress of their discovery process in real time.

Document Reviews

With Eclipse, a thorough and efficient review process is possible. Administrators can:

  • Distribute documents for review to individual reviewers or to groups of reviewers.

  • Manage workflow through the use of multiple review passes and real-time dashboard analytics.

  • Monitor the progress of cases and establish real-time alerts.

Document Relationships and Boundaries

Eclipse allows extensive document information for users. Administrators can define the level of detail to be included in a case, so that users can:

  • View document boundaries.

  • View document family relationships.

Export/Production

Eclipse administrators can efficiently:

  • Export or produce image files as single-or multi-page TIFFs or PDFs (optionally searchable).

  • Include placeholders for missing images.

  • Export or produce native files and/or single- or multi-page text files.

  • Endorse documents.

  • Use Production Shield to help eliminate production errors.
