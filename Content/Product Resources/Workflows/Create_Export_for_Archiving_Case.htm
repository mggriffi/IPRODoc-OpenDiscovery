Create Export for Archiving Case

This workflow is used when a case within the multi-tenant environment is being removed/deleted from IPRO Services.

Note: Data will be retained for 30 days, at no additional charge, before full deletion from IPRO’s environment.

Archive Definition

All metadata, natives, text, and images are exported together with a CSE file.

Includes:

  • CSE file (case template file) - may be used to recreate case in Ipro products, includes the following:

    • Saved Searches

    • Database Fields

    • Tag Palette

    • Redaction Palette

    • Noise Words

    • Number of Indexes

    • Coding Forms

    • Relationships

    • Date formatting options

  • DLF, DAT, LFP & OPT files for importing the following:

    • All fielded metadata, including coding work product and tags

    • Images - single-page Tiff

    • Native files

    • Text files

  • Data stored on FTP in the Productions folder.

Not Included:

  • Keyword Lists

  • Review Passes

  • TAR Projects

  • Production History

  • Document History

Export Case for Archival

The following steps must be completed:

  1. Prepare export of case data. For detailed instructions, see Export Checklist for Archives.

  2. Submit request to Cloud Services (services@ipro.com) to provide CSE file and remove case.

  3. Download from FTP (Media Shuttle) Production data for case.

  4. Save the CSE file, Export files (DLF, DAT, LFP, OPT) and Production data for case.

Bring Case Back Online

Depending on the size of data, timing to bring case back online is 1-3 days.

  1. Request a new Case to be opened with Cloud Services. See Cloud Services: How to Request a New Case.

  2. Provide the following to the Cloud Services team:

    • Data load files: DAT, LFP, OPT, DLF

    • CSE file

    • Production Data downloaded from FTP (Media Shuttle)

  3. Deliver data to the Cloud Services team, using IPRO’s file transfer portal Media Shuttle. See Using Media Shuttle to Transfer Files.

Note: After you upload data to Media Shuttle, you must submit a ticket or send an email to services@ipro.com letting them know that data has been uploaded to Media Shuttle for a specific project. No specific format is needed; simply include the project name and the file name in Media Shuttle.
