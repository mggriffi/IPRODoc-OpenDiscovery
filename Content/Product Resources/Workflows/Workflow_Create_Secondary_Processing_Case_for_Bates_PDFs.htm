Workflow: Create a Secondary Processing Case for Bates PDFs

As an administrator of it is common to receive already processed data in the form of PDFs named per Bates Number. This document will walk through setting up a secondary processing case to specifically handle Bates Numbered PDFs without a load file. This processing case will be used to extract text from the PDF and perform OCR if the PDF is not already searchable. The PDFs will be loaded into review named per the beginning bates number (filename).

Note: This procedure is assuming you already have a processing and review case created through Case Management.

Step 1: Create a New Processing Case

  1. Log in to and navigate to Case Management.

  2. Click the hamburger icon under Code Name to add a new Case to an existing Code Name Group.

  3. Create a new Processing Case with an identifying suffix of ‘ – PDF’

  4. Your Code Name group should now consist of two processing cases and one review case.

Step 2: Create a New Import Series

  1. Open Case Management and locate the Processing case for which you would like to create an import series.

  2. Click the hamburger icon corresponding to the required case.

  3. In the menu that displays, select Case Settings. The work area opens. Select the Load to Review tab near the top.

  4. On the Review Importing page, select Create Import Series.

  5. On the first step of the Import Series wizard, define a name for the new Import Series, such as "Bates PDF."

    Note: Ensure the Set as Active Series option is selected. When active, the import series is able to promote ingested documents into the selected Review case. When inactive, the import series is "turned off" and any customization defined in the series is not used during processing.

  6. Proceed through the steps of the Import Series wizard, customizing as needed. See Create an Import Series for more information on the various options.

  7. On the fourth step of the wizard, ensure you select the Use filename option.

  8. When all options have been defined, click Finish to create the Import Series.

Step 3: Ingest Data in Self-Service

After the above configurations have been completed you are ready to ingest the data through Self-Service.

  1. Open the Self-Service module.

  2. On the Self-Service page, click I want to upload a file to process into review.

  3. Select the file you want to upload and click Open.

  4. Select your new PDF processing case.

  5. Select Custodian > click Go.
