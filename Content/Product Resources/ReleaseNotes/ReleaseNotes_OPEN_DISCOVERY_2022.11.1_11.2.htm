Release Notes 2022.11.1 and 2022.11.2

The new release of OPEN DISCOVERY focuses on the features you have been asking for. IPRO is committed to an ongoing balance between innovation and meeting customer needs.

Schedule your upgrade with IPRO Support today!

New Features

Review

Faster Document Review – Improved performance on Document Viewer allowing documents to load, on average across all data types, twice as fast. *Upon upgrade completion, automation to re-optimize all cases will go through job manager. To avoid normal priority jobs from being delayed in job manager queue, please adjust agent priority to 50% until completion.

Print Directly from Image Viewer – Save time with the new option available to print directly from the Image and Production Viewer tabs to PDF.

Improved Keyword List Management

  • Document Search Scope can now be tailored down to a smaller set of documents by using a saved search to target specific document sets. This allows users to create custom keyword list and hit reports for custodian(s) and\or data set(s).

  • New Simplified and Modern Interface, improved usability, and consistent interface experience with the rest of the application.

Native File Download from Icon - Users using the document grid can click on the file-type icons to directly download the file, avoiding extra and unnecessary clicks to review native files.

Hot Key Legend Update - Updated hotkey legend to include the newly added "Ctrl + Backspace" to delete the selected redaction.

Document Grid Update – “Select All” or "Clear Selected” ability added to the top of the document grid to make it easier for users to interact with. Users can now use these options when preparing documents for mass actions to quickly add all or remove document selection.

Resolved Issues

Review

AR-16592 – Improved the optimization process to account for instances when replacing native files that were previously optimized. This will trigger the ability to re-optimize the new documents overlaid.

AR-16994 – Improved performance when entering new case and initial loading of Cluster Wheel.

AR-16488 – Improved handling for redaction deletion when mouse activity is used simultaneously to avoid auto-selecting and resizing newly selected redaction.

Processing

AP-5485 – Updated intended workflow to rollback bates numbering for last deleted Data Extract Export job from the controller.

AP-5482 – Improved handling of page level OCR for duplicate files when the original file stored a value of incomplete OCR.

AP-5439 – Improved handling when creating a new Process Export Series from eCapture controller.

AP-5409 - The password protected file identification routine has been improved to ensure that the 'Password Applied' flag is only used when the password was successfully applied.

AP-5452- Imaging process enhancement to better handle Office file types when using IPRO Q if they weren’t originally ingested with eCapture.

AP-1761 - The 'Streaming Detailed Dedupe Report' would throw a collation exception for SQL environments configured with a collation other than 'SQL_Latin1_General_CP1_CI_AS'. This has been enhanced to support any non-default collation setting.

AP-5562 – Improved process in recognizing attachments to EML email files during Legacy Discovery.
