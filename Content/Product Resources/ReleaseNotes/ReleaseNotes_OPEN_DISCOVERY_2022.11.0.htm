OPEN DISCOVERY Release Notes 2022.11.0

The new release of OPEN DISCOVERY focuses on the features you have been asking for. IPRO is committed to an ongoing balance between innovation and meeting customer needs.

Schedule your upgrade with IPRO Support today!

Note: This release is currently only available for IPRO Cloud customers.

New Features

Review

Redaction Enhancement - The "x" used to remove a redaction has been enlarged and standardized to grow and shrink with a redaction making it easier to find on the viewer with proportional scale.

Redaction Deletion – Use CTRL+backspace key to delete redactions once highlighted. Improving the user experience with an easily remembered quick key that allows a reviewer to stay fully in the redaction interface without having to move off page.

Page Count Calculation - Total page count can now be calculated for selected documents that have images. Standardizing the interface and providing page counts beyond text-only documents to all documents.

Enhanced Document Navigation – Document navigation streamlined and more intuitive to get reviewers up to speed faster.

  • Updated "Save & Next" button replaced with an “Arrow” to function as a save and move to the next document while taking up less real-estate on screen.

  • “Next” moves to the next document without implied save. If the document was updated, the user will be prompted with a message that changes will not be saved if they continue.

  • “Back” moves to the previous document without implied save. If the document was updated, the user will be prompted with a message that changes will not be saved if they continue.

  • "Undo" removes any unsaved changes if changes exist.

  • "Save" will save changes to the document.

Improved Document Grid – New and improved document grid to increase usability, visualization, and filtering options, for a more friendly user experience.

  • Column width changes made by a user are maintained from session to session. No more realigning each column at the start.

  • Select all, selects all documents in the active search versus the visible span by default.

  • Shift-Click enabled to select multiple documents within the grid providing greater consistency with expected application behavior.

  • The filtered document count currently selected is now fixed to the bottom left corner of the grid providing unobtrusive persistent visual cues.

  • The Applied relationship is actively visible across all instances when in the document grid.

  • Multi-Sort by holding the "ctrl" key when applying filters provides intuitive access to multi-sort and can be applied across up to 9 sort fields.

  • The column name field now has a direct text filter, enabling reviewers to quickly filter on names without having to run a custom search.

Enhanced Transcript Review to increase visualization and usability.

  • New option for mouse hovering over Page: Line to preview the context.

  • Displaying issues in alphabetical order within the report wizard.

  • Page: Line information is now available when creating or editing annotations.

  • Toolbar redesign to simplify view of controls and increase accessibility with labels.

  • Updated and easier user interface for navigation to Transcripts, Reports and Search.

Persistent Navigation within Tags - You can now collapse or expand tag groups as needed and will persist as you navigate between documents providing a higher level of usability and a more intuitive user experience.

Search for Tagged & Un-tagged documents – In document viewer, users can create a search to quickly find all documents where a specific tag has been applied via right-click option from the tag. The search criteria are added to the queue in the Visual Search dashboard and can be run as is or updated with additional search criteria.

File Size Limit – Reducing the mixed messages of small file sizes that do not render on the screen, the file size limit for loading into the viewer is now calculated using the html rendering rather than the native file size. The messaging has been updated to be more descriptive and user friendly.

Relationship Mass Update - Mass action buttons are easier to find as they now persist at the top of the viewing area. Furthermore, the position of the coding fields and tags have been swapped to make the tags more accessible when mass tagging, reducing scrolling time and the potential for user error.

Renaming of Saved Searches – Existing saved searches can now be renamed by updating the title of the search. The updated search saved will either overwrite the existing search or create a new search with the new name, saving time and improving reviewer efficiency.

System Management

Enhanced Public API – IPRO API now allows you to leverage new options to secure automation and integration into your customized workflows.

  • Reporting – Trigger reports from your applications, e.g., your ingestion report is available within your personalized tracking tool without logging into OPEN DISCOVERY.

  • OCR – Additional API extensions for OCR. Extensions enabled to send feature flags for IPRO supported language types and point to existing images. Additional features enabled to return OCR STATUS, DOCID, FILENAME, TEXT, TEXTPATH, OCRCONFIDENCE, SUCCESS, and ERRORTEXT fields from the API. Enabling closer integration to external systems and options for advanced automation in reporting or data loading.

Custom Product Branding – Customize the web application user interface by replacing IPRO logo with your own personalized logo for branding the UI with your company name.

Manage LIVE EDA Connections – You can now manage multiple LIVE EDA connections at the client level instead of the environment level to support multiple clients from one dashboard enabling multi-site and multi-client management from a common interface.

Transcript Management Permissions – Customize transcript permissions by allowing user groups to view, import or delete transcripts based on group permissions.
