New Features

Review

Scrolling Navigation Within Web Viewer - Page navigation within the Web Viewer (Native) now supports scrolling. Users can use the scroll, page forward\back and highlighting navigation buttons interchangeably to navigate through a document within the Web Viewer tab.

Support For Ingestion And Review Of RSMF Output - Users can now ingest chat data that has been converted to the RSMF format through Open Discovery and display those conversations in Review.

*NOTE* - Live EDA Users also have the added benefit of integrated support for promoting chat data in RSMF format into Open Discovery.

Improved Consistency Across Document Viewer Tabs - Page navigation controls and layout within the IMAGE and PRODUCTION tabs of the Document Viewer are now consistent with the experience found in the WEB VIEWER tab.

Option To Disable the Redaction Drawing Tool When In Use - A new button is present to the right of the dropdown allowing the user to disable the redaction drawing tool so that other features may be used within the document viewer. Selecting the redaction category dropdown turns the drawing tool back on.

UI Overhaul To Export Sets - The UI for export sets has been overhauled to be more modern visually without losing any functionality previously available.

Resizable Relationship Panel - Users can now resize the relationship panel when pinned to show all fields and tags available on each relationship.

UI Updates To Keyword Management -The UI for Keyword Management has been overhauled to be more modern visually without losing any functionality previously available.

Display Active Users Report - Case Administrators have the ability to see all active user accounts logged in and working within review. For more information click here.

Mass Action Download Of Native Files - Users can now quickly download native files with a load file without having to run an export. We made it quick for users to quickly download and send native files with a load file as a new mass action item called, “Download”.

*NOTE* - Native file downloads are limited to 1,000 native files. Anything larger is recommended to use Export Sets functionality.

This is a permission-based feature:

  • Exporting native files is tied both to the permission “launch native” and permission “mass export”.

  • Exporting metadata is tied only to the “mass export” permission.

Use Existing Case As Template During Case Creation - Case creation has never been easier, now you have the ability to point to an existing case to use as a template for both Processing and Review. All settings supported by INI and CSE files are supported in this feature. Eliminate managing files on disk by building template case(s) all in Open Discovery web interface.

  • Settings included in CSE supported by this feature:

    • Tag Palette (tag groups and tags)

    • Database field definitions (field name, type and security)

    • Redaction Palette

    • Relationship Definitions

    • Coding Forms

    • Saved Searches

  • Settings included in INI supported by this feature:

    • Ingestion Settings

    • Filtering Settings

    • Imaging Settings

    • Import Series Fields

Updated NIST DB Update Utility - An updated NIST DB utility that supports the change to SQLite database format of the NIST NSRL hash set.

*NOTE* - This utility is available for download from MyIpro and is also included in the 2024.11.6 VM image for convenience.

Update To Support Relativity Promotions When Using Relativity Server 2023+ - Updates to continue supporting promotions to Relativity Server 2023

Important Configuration Changes Required

The Relativity Server URL has changed and will require updating to continue promoting into Relativity Server 2023.

  • The Relativity Server entry will need to be updated as follows: https://relativity/Relativity.Services.

  • Now it must be updated to match the following: https://relativity/Relativity.REST/api.

SQL Server 2022 Open Discovery Certification - SQL Server 2022 has now been thoroughly tested for use with Open Discovery.

Resolved Issues

Review

AP-5885 - Improvements have been made to scale redaction text so it fits horizontally in the defined area, then scale the text down if it doesn’t fit vertically so that long, narrow redactions are properly burned in during production exports.

AR-10157 - Improved Picklist coding to allow for partial searching and restored the ability to scroll through entire list.

AR–18845 - Printing documents to PDF with slipsheet is improved to include tags on slipsheet.

AR–1960 - In some instances, imported images could display with colors inverted. A correction has been made to better handle these when they are encountered.

AR–12018 - Updated Print to PDF field endorsement to not require “Mass Edit Fields” permission to endorse fields on an image.

AR-19321 - Restored Tally visual search chip to quickly allow user to change the selected value without having to run a new tally search.

AR-18173 - Updated Hide/Show menu from document grid to display or change the order of columns once all columns are selected in a case.

AR-18138 - Updated Mass Export to CSV file from document grid to better handle incorrect spacing with commas.

Search

AR-18852 - Updated Random Sampling UI to not be cutoff if Text and Field Search selection is expanded.

AR-18759 - The date selector could be cutoff in certain situations when configuring field searches involving date fields. Improved handling has been introduced to prevent the date selector from not being fully displayed.

AR-19073 - Updated Advanced Search to not require "Manage Keyword List” permission to run searches for Keyword terms.

AR-17587 - Improved UI to handle split view with saved search area pinned to not cut off Advanced Search criteria window.

Administration

AR-10422 - In some instances, deleting records from a search in Administration and navigating away could cause the incorrect documents to be deleted.

AR-10913 - Improved handling for deleted or inactive cases to also remove the scheduled index jobs to run.

PS-9634 - When the jobs database reached a certain size, timeouts would begin to be encountered. Updates to the stored procedure to gather these results have been improved and indexes added to the database to support much larger counts and improve performance.

AR-19200 - Updated case creation to not fail if case settings file has .CSE extension in all caps.

AR-20039 - Improved handling when running non-normalized clustering to not require a text file for all documents.

AR-15138 - Transcript importing improved to handle PTF header to properly identify volumes and page numbers.

AR-15498 - Updated Product \ Export Validation report to properly show “Missing Priority Documents =Yes” if not all documents in the search have an image priority selected. Prior versions were reporting “Missing Priority Document = No” with a conflict to override.

AR-12967 - Added additional clean up automation to C:\Windows\Temp DIR to prevent images and natives from not displaying.

Reports

AR-18474 - Updated Review Case Summary report to show Optimized Native size in MB format.
