discovery ][ Enterprise Release Notes 2022.10.0

IPRO continues to progress discovery ][ Enterprise with a new release that focuses on Transcript Management, Public API access, Export, and Performance enhancements, compliance, and much more.

Please contact IPRO Support to schedule your upgrade today!

New Features

System Management

Public API - Leverage the IPRO API for powerful and secure automation and integration into your customized workflows. This release includes the Enterprise Public API, which secures and centralizes access to the available API and allows a host of capabilities from Case Management to Ingestion, Exporting, Users, Groups and Permission, and others that allow integration with Enterprise. See Overview: Public API.

FIPS Compliance - FIPS are standards and guidelines for federal computer systems that are developed by the National Institute of Standards and Technology (NIST) and approved by the US Secretary of Commerce. Enterprise is now FIPS compliant when enabled.

Review

Transcripts - The new transcript viewer allows reviewers to import, view, and annotate transcript files.  Easily tag elements of a transcript with issue codes from your existing review case or create new issues for use on transcripts. A word index provides visibility into frequency and location of words or topics. Once annotated, a report on the elements that contain issues in a variety of output formats is created. A new security permission (View Transcripts) allows SuperAdmins to control group permission to view and administer transcripts. See Overview: Transcripts.

Export and Production Enhancements - Export set configuration has been updated in Review to allow enhanced configuration of Native and Image output. Users can identify specific documents that will receive Native and\or Image output based on a document tag. Pre-existing functionality to identify based on file type is still in place, while providing another export preference to allow for customized exports. See Workflow: Create a Production.

Change Request: Review Template - The following updates to the review template were added to help implement best practices when upgrading to the latest release. Template changes will only apply to new cases.

  • New Tags created for Export and Production Enhancements – “Output Native” and “Output Image” tags were added to existing Production Tag Group to identify output type for documents being exported or produced from a tag.

  • New Saved Search folder “Production” with saved search “Output Native” and Output Image” were created to help identify documents tagged with “Output Native” and Output Image”.

Performance Enhancements – Documents display faster and enable case admins with a more granular level of control in how they optimize or pre-render documents that get displayed within the web and image viewers. See Manage Performance.

Change Request: Review Template - The following updates to the review template were added to help implement best practices when upgrading to the latest release. Template changes will only apply to new cases.

  • New Saved Search “Optimization Scope” under existing Admin folder created to identify documents that need performance optimized. File Extension equals PDF, PNG, JPEG, JPG, TIFF and TIF, when File Size is greater than 1000 KB.

  • Performance Management is now defaulted to ON, and scope includes new saved search “Optimization Scope” and Image Settings set to First Page Only.

Processing

Review Import Settings – Import into review settings has been implemented to streamline and support Ingestion of documents to IPRO Review from the new web interface. Import Series configuration allows field mapping, numbering, and other settings relevant to getting documents into Review. See Overview: Import Series.

Resolved Issues

Review

AR-12818 – Keyword List searches being modified on search edit now retain previous tags selected.

AR-13103 - Tag group sort order has been restored back to the order determined within the coding form.

AR-13012 – Optimized cluster processing queries to improve performance when building new clusters.

AR-12746 – Updated viewing of documents from batch or pre-generated HTML files from prior application versions to now be viewable without throwing an error.

AR-11851 – Updated Active Learning-disabled review passes to be available right away for checking out on case enter screen.

AR-12669 - Using an Active Learning-disabled review pass no longer prevents users from checking out a batch right away while the index is still building.

AR-11753 – Updated saved search sub-folders sort in alphanumerical order like searches in top level folders.

AR-11017 - Fixed an issue where long and narrow redactions would not have their redaction text re-sized to fit the resulting redaction.

AR-13283 – Reduce the size of the saved search query to eliminate browser crashing.

AR-14098 - Updated Review Pass Management to properly identify new documents to be added during "Create Batch" so they are added to the existing review pass.

Processing

AP-4917 - In some situations an OCR Confidence value previously could exist when OCR is being performed resulting in the Worker looping on the task. Handling has been implemented to prevent looping to complete the task and avoid delay in job completion.

AP-4781 - Emails containing Arabic text using the internet code page 'windows-1256' rather than Unicode previously could result in missing extracted text. Handling has been implemented for this very specific scenario to ensure the proper text is extracted.
