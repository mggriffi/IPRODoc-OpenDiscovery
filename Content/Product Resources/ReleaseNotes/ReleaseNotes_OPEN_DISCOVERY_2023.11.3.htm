OPEN DISCOVERY Release Notes 2023.11.3

The new release of OPEN DISCOVERY focuses on the features you have been asking for. IPRO is committed to an ongoing balance between innovation and meeting customer needs.

Schedule your upgrade with IPRO Support today!

Note: This release is currently only available for IPRO Cloud customers.

New Features

Review

Improved Image on the fly – Improvements to Image on the fly (IOTF) output for enhanced quality, formatting, and file type support.

Document History Filtering - Added the ability to filter by action type or user when reviewing audit logs within document history view.

Assign and Un-Assign Batches (Non-Active Learning Review Pass) – For Non-Active Learning Review Passes, we’ve added the ability to assign or un-assign batches to individual users in the batch management screen. Assigned batches will be prioritized as the first batch to be viewed when user enters a review pass.

Sticky "fit" in Viewer - Fit to Width and Fit to Height option are now set to stay in-place as users navigate through either the native web and image viewer.

Auto Scale Native Viewer & Image Smoothing – Native and Image Viewer has been updated to dynamically scale viewer based on best fit within viewer window. This improves users experience when trying to read content, apply actions, zoom, and eliminates un-used space.

Change Request

Case Creation Template Update – Optimization Scope Saved Search

  • Removed File Type Filter to allow for all files to be optimized.

  • Removed documents that are flagged as Protected or Password Protected from being optimized.

  • Remove native documents larger than 30 MB from being optimized as they should be reviewed in native format.
