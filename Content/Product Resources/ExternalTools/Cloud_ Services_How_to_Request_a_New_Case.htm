Cloud Services: How to Request a New Case

New case requests are performed through Cloud Services. The steps are as follows:

  1. Select Case Request to create a request ticket.

  2. Firm or Company name: Please enter the name of your Firm or Company as you are known by IPRO.

  3. Case name: (as filed with the court): We suggest you register the case name in your IPRO environment with the same name under which it is filed. Strict parity is not a requirement, however, a formal case name is recommended. In the next field you will have the opportunity to give your Review Database database a nickname or code name.

  4. Review Database Name (Nickname):This is the internal naming convention you will give to your Review database.

  5. Docket Number:This an optional field, but we recommend that you include this information for better information governance.

  6. Time Zone:Enter the time zone where your review will be taking place.

  7. De-Duplication:You can set your deduplication preferences here. You have three options:

    1. Custodian Deduplication: This setting will remove exact duplicates that appear within the entire dataset associated with a single custodian.

    2. Case Level Deduplication: This setting will remove exact duplicates that appear across the entire dataset in your matter.

    3. No Deduplication: This setting will not remove any exact duplicates from your review.

  8. Notes: This field is for additional information about your case.

    1. Adding/Removing Users: List out the users for this case by Name + Email Address. Please include the permission level requested for each user. In Starter and Business Packages, you have the choice of either Review or Admin level permissions.

    2. You can enter any notes that you would like to make regarding the setup of your case ticket such as: Case Data already uploaded to Media Shuttle, Brief Description of specific templates, etc.

  9. Attachments: Occasionally, there may be a need to share some documentation about your matter with your Cloud Services Team. You can upload that here.

    Note: This is not where you will be uploading the review data for your Cloud Services Team.

  10. Click Send.
