Supported Native File Types

Native files (such as emails, spreadsheets, or word processing documents) included in a case or project can be viewed in the following products. These products utilize Oracle® Outside-In Technology to identify file types and present the native files. Over 500 file types are supported, ranging from common Microsoft® applications to legacy files.

Product and Version

The following products use Oracle Outside-In Technology 8.5.2:

  • Eclipse® by 2016.3.0 and later

  • eCapture® by 2015.2.0 through 2017.3.8

The following tables contain the supported file types according to software categories.
