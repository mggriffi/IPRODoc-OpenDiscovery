IPRO Cloud Portal: Reset a Forgotten Password

If you forget your IPRO Cloud password, you can reset it using the instructions outlined below:

  1. Navigate to the IPRO Cloud portal and click the Forgot Password link, below the Sign in button.

  2. On the Forgot Password page, enter your user ID and click Submit.

  3. On the Authentication Method page, click the Send Me a Push button.

    When the push notification is sent the blue bar appears.

  4. A notification is sent to your phone asking if you are attempting to log in. On your phone, click the Green check mark.

  5. In the browser, two of the three security questions you selected when you first registered your account are displayed. These questions are randomly selected. Correctly answer the two selected questions. Responses are case sensitive. An example of two selected questions is shown below.

  6. Click Submit.

  7. The Change Password page displays. Create a new password following the requirements listed below.

  8. When finished, click Submit. After the password is saved ,the following message displays:

  9. Click the Log On button to proceed to the Sign In page where you will enter your User ID and newly-created password.

Note: To set new questions and answers, please contact Support for assistance.

Password Complexity Requirements

Note: If you attempt to set a password that does not meet the following complexity requirements, the unmet requirements will be listed in an error message from the server.

Password Complexity Requirements:

  • At least eight characters long
  • At least one character from at least three of the following four categories:
    • Upper case
    • Lower case
    • Base-10 digits
    • Nonalphanumeric (~!@#$%^&*_-+=`|\ (){}[]:;'"<>,.?/)

    Note: The password must be unique from your 5 prior passwords.
