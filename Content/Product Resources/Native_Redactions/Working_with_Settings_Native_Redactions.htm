Working with Settings: Native Redactions

The settings containIPRO uses an advanced native file redaction platform with highly efficient features to perform complex redactions within excel documents. It supports point-n-click manual redaction as well automated mass redactions based on provided criteria.

Settings

From the toolbar, click the down arrow to open:

PII & Keyword Explorer:

This form summarizes PII information across all the documents. You can select documents from the list for mass redactions.

 

Term Group Explorer:

You can create a group of terms with multiple terms in it. Each sheet will be identified when any of the terms are present and will display the sheet with unique matching terms count. Double-clicking on any document from the list will open it in the Viewer.

 

Visually Similar Sheets Explorer:

The data may be the same or different across the sheets in a group, but the look and feel are similar. This tool creates groups of sheets based on that similarity. It also facilitates applying redactions on all sheets in a group in a certain area.

 

Thread Explorer:

Excel file threading is similar to email threading. This is when the Excel file is fully contained within another Excel file and is listed underneath of the larger Excel files.

 

Cluster Explorer:

Excel file clustering is based on similar sheets. If two or more documents have at least one sheet that is a duplicate across them then they are placed in a cluster. This process will continue to look for any documents that contain duplicate sheets with any of the files in the cluster. These files are also added to the cluster.

 

Duplicate Sheet Explorer:

It assists in finding duplicate sheets in the document set. Redactions propagate across by default so they only need to be applied once, not separately. Results will appear in the right side pane by default.

 

Sheet List Explorer:

In this explorer, you can create lists of the sheets to make the document review more efficient.

 

Hidden Content Explorer:

This form lists all of the documents which have original sheets hidden and password protected. It also lists all of the documents which have data hidden by making the font color and the background color the same. Double-clicking on any document from the list will open it in the Viewer.

S

 

Pseudonym Explorer:

This form allows users to apply masking for each value in a value list. E.g. email john@company.com is replaced in the export process by PRSN0001 and email tom@newcompany.com will is replaced by PRSN0002 etc. This retains the entity-relationship between different values while redacting them.
