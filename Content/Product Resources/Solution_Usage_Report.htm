myIPRO Solution Usage Report

The my Solution Usage Report can be accessed from my here. Follow the steps below to run the report.

  1. Log into https://my.ipro.com/client to access the report.

  2. Select Reports from the menu.

  3. On the Solution Usage Report page, enter the Usage Period and the Case you want to generate the report for.

  4. Enter a dollar value per gigabyte for the processing and review items on the report.

  5. Preview or download the report.

    The report is generated.

    Note: The Processed Data Size is calculated based on the job date instead of averaging all time. Review Data Size is calculated based on averaging all time.

Report Details

The Solution Usage Report contains the following details:

item

definition

Case

The case name or the referred to code name.

 

Review Data Size

 

The average of all hosted review data including all images, natives, extracted text and production images. All data hosted is pulled regardless of when an import or production was done. This does not include database size.

Processed Data Size

A peak sum of the processed data including discovery, extracted data and processing jobs. This will be based on the job completion dates. Only jobs done within the date range are considered for calculating the peak sum. This does not consider the source data (compressed) size and does not include database size.
