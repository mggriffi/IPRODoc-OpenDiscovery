Add Media to Your Delivery

After a delivery is created, the media associated with the delivery can be defined and added to the delivery to which it belongs. Media can also be defined separately from deliveries, with the option to add them to a delivery later if needed. For more information, see Overview: Deliveries.

There are two options for adding media to your delivery. You can create new media that will be added directly to your delivery, or choose from existing media to add to your delivery.

Create New Media

In Media Manger, you can create new media from the Deliveries tab or the Media tab.

To create media that will be added to a delivery, use the Deliveries tab:

  1. On the Deliveries tab, double-click the delivery to which media is being added.

  2. On the Delivery summary page, click Add New Media.

  3. Complete the Add New Media form. Ensure you have all necessary details and files before proceeding. See Create New Media for details.

  4. If this is for an outgoing delivery, the data can now be shipped to clients, opposing counsel, etc. If this is for an incoming delivery, continue with Process Data Locations and Copy Data to Network.

Add Existing Media

After a delivery has been created, complete the following steps to add existing media to the delivery:

  1. In Media Manager, click the Deliveries tab.

  2. Locate and double-click the delivery to which media is being added.

  3. In the Current Media in Delivery area, click Add Existing Media:

  4. In the media list, locate and click to select all media to be added to the delivery.

  5. Click Add Selected Media.

  6. If this is for an outgoing delivery, the data can now be shipped to clients, opposing counsel, etc. If this is for an incoming delivery, continue with Process Data Locations and Copy Data to Network.
